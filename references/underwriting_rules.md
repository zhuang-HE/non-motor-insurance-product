# 非车险核保规则与风险评估

> **最后更新**：2026-03-30  
> **适用范围**：非车险产品核保全流程规则

---

## 一、核保基本原则

### 1.1 核保三原则
1. **风险可控原则**：只承保可评估、可控制的风险
2. **合规经营原则**：严格遵守监管要求和公司制度
3. **效益导向原则**：确保业务质量和盈利能力

### 1.2 核保权限
> 标准表述："分级授权、从人授信、一人一码、严禁转用"

| 权限等级 | 保额范围 | 审批层级 |
|---------|---------|---------|
| 一级核保 | ≤100万元 | 支公司/营管部 |
| 二级核保 | 100-500万元 | 分公司 |
| 三级核保 | 500-5000万元 | 分公司总核保人 |
| 四级核保 | >5000万元 | 总公司 |

## 二、投保人/被保险人风险评估

### 2.1 基本资格审查
- [ ] 投保人具有完全民事行为能力
- [ ] 营业执照有效、统一社会信用代码正确
- [ ] 经营范围与保险标的相符
- [ ] 注册资本符合要求
- [ ] 成立时间超过1年（或符合特别规定）

### 2.2 信用状况评估
| 评估维度 | 检查内容 | 风险等级 |
|---------|---------|---------|
| 征信记录 | 人民银行征信报告 | 不良记录 → 拒保 |
| 诉讼记录 | 企查查/天眼查 | 重大未决诉讼 → 加费/拒保 |
| 保险记录 | 行业共享平台 | 欺诈记录 → 拒保 |
| 财务状况 | 财务报表/纳税记录 | 连续亏损 → 审慎承保 |

### 2.3 行业风险评估
| 行业类型 | 风险等级 | 核保要点 |
|---------|---------|---------|
| 轻工业 | 低 | 标准流程 |
| 重工业 | 中高 | 重点关注设备老化、环保风险 |
| 化工 | 高 | 需安全评估报告、环保合规证明 |
| 电子信息 | 中 | 专利有效性、研发阶段 |
| 生物医药 | 中高 | 临床阶段、合规认证 |
| 食品加工 | 中 | 食品安全许可、卫生条件 |

## 三、保险标的风险评估

### 3.1 风险查勘要求

#### 必查项目（满足以下任一条件）
- 保额超过500万元
- 属于高风险行业
- 历史赔付率高
- 特殊标的（古董、艺术品、精密设备）
- 核保人认为必要

#### 查勘内容清单
- [ ] 标的实际状况（物理检查、拍照）
- [ ] 周边环境风险（地质、气象、邻近危险源）
- [ ] 消防设施检查（灭火器、喷淋、报警、通道）
- [ ] 安全管理评估（制度、培训、演练）
- [ ] 历史损失了解（近5年损失记录）
- [ ] 防护措施评估（防盗、防水、防潮）

### 3.2 风险分级
| 风险等级 | 判定标准 | 承保策略 |
|---------|---------|---------|
| 优质（A级） | 行业龙头、管理规范、无历史赔付 | 标准费率甚至优惠 |
| 标准（B级） | 经营正常、管理基本到位 | 标准费率 |
| 关注（C级） | 管理有缺陷、偶有损失 | 加费10%-30% |
| 限制（D级） | 高风险行业、管理混乱 | 高额加费或拒保 |

### 3.3 特殊标的评估
| 标的类型 | 评估要点 | 参考标准 |
|---------|---------|---------|
| 高风险标的 | 独立评估报告 | 专业鉴定机构 |
| 古董艺术品 | 来源合法性 + 专业鉴定 | 国家文物鉴定标准 |
| 贵重设备 | 品牌型号 + 维修记录 | 厂商报价/二手市场 |
| 知识产权 | 有效性 + 维权可行性 | 专利局查询 |
| 数据资产 | 价值评估 + 安全等级 | 数据资产评估标准 |

## 四、承保条件审核

### 4.1 保险金额审核
- [ ] 保险金额不超过保险价值（不足额投保比例）
- [ ] 保险金额不低于最低承保要求
- [ ] 分项保额设置合理
- [ ] 保额与风险实际匹配
- [ ] 高保额业务已安排再保险

### 4.2 免赔额设置
| 免赔类型 | 设置原则 | 参考标准 |
|---------|---------|---------|
| 绝对免赔额 | 根据损失频率设定 | 财产险：1万-50万 |
| 绝对免赔率 | 根据损失幅度设定 | 财产险：5%-20% |
| 相对免赔额 | 较少使用 | 特殊约定 |
| 复合免赔 | 两者取高 | "每次事故2万或损失金额10%，以高者为准" |

### 4.3 保险期间审核
- [ ] 保险期间明确（标准1年期）
- [ ] 起保日期合理
- [ ] 短期保险符合短期费率规定
- [ ] 工程保险期间覆盖施工期 + 试车期
- [ ] 科技保险期间与项目周期匹配

### 4.4 特别约定审核
- [ ] 特别约定内容合法合规
- [ ] 不与主条款冲突
- [ ] 不损害被保险人合法权益
- [ ] 表述清晰无歧义

## 五、再保险安排

### 5.1 自留额管理
| 业务类型 | 自留额参考 |
|---------|-----------|
| 财产险 | 单笔保额500万-2000万 |
| 责任险 | 单笔限额1000万-5000万 |
| 工程险 | 单笔保额1000万-5000万 |

### 5.2 再保险方式
| 方式 | 适用场景 | 操作要点 |
|------|---------|---------|
| 成数合约 | 统一处理大量业务 | 固定分出比例 |
| 溢额合约 | 大额风险分散 | 设定自留额+线数 |
| 临时分保 | 特殊/高额风险 | 逐笔协商 |
| 超赔合约 | 巨灾/大赔案保障 | 设定起赔点+限额 |

## 六、反洗钱审查

### 6.1 客户身份识别
- [ ] 自然人：身份证/护照原件核验
- [ ] 法人：营业执照、法定代表人身份、实际控制人信息
- [ ] 异常投保行为识别（现金交易、频繁退保、保额异常）

### 6.2 可疑交易报告
| 可疑特征 | 处理方式 |
|---------|---------|
| 大额现金支付保费 | 报告合规部门 |
| 无法提供合理投保目的 | 加强审查/拒绝承保 |
| 保险金额与投保人经济状况严重不符 | 要求补充说明 |
| 短期内频繁投保退保 | 记录并报告 |

---

## 七、决策规则

### 7.1 承保决策矩阵
| 条件 | 决策 |
|------|------|
| 所有检查项通过 + 风险B级以上 | ✅ 同意承保 |
| 部分检查项不通过但可整改 | ⚠️ 附条件承保 |
| 核心风险不可控 + 无有效缓释措施 | ❌ 拒绝承保 |
| 超出核保权限 | ⬆️ 上报审批 |

### 7.2 加费/拒保标准
| 情形 | 处理 |
|------|------|
| 风险C级 + 可缓释 | 加费10%-30% |
| 风险D级 + 不可缓释 | 拒保 |
| 不良信用记录 | 拒保 |
| 保险欺诈记录 | 拒保 |
| 超出公司承保能力 | 拒保或临分安排 |

## 附录D：反洗钱审查清单

### D.1 客户尽职调查（CDD）
- [ ] 客户身份信息采集完整
- [ ] 实际受益人识别
- [ ] 经营目的和资金来源核实
- [ ] 客户风险等级评定

### D.2 强化尽职调查（EDD）
以下情况需执行强化尽职调查：
- [ ] 政治公众人物（PEP）相关
- [ ] 高风险国家/地区关联
- [ ] 复杂所有权结构
- [ ] 异常交易模式

### D.3 交易监控
- [ ] 大额交易报告（单笔≥5万元或等值外币）
- [ ] 可疑交易报告
- [ ] 定期回访和高风险客户复核
