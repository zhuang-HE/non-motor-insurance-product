## 一、条款用语标准（完整版）

### 1.1 保险责任表述

| 责任类型 | 标准表述 |
|---------|---------|
| 财产损失 | 对因下列原因造成保险标的的损失，负责赔偿 |
| 第三者责任 | 对被保险人在保险期间内造成第三者的人身伤亡或财产损失，负责赔偿 |
| 产品责任 | 对因产品存在缺陷造成使用者或第三者的人身伤害或财产损失，负责赔偿 |
| 雇主责任 | 对被保险人雇用的员工在受雇期间因工作遭受意外事故，负责赔偿 |
| 公众责任 | 对被保险人在经营场所内因意外事故造成第三者人身伤亡或财产损失，负责赔偿 |

### 1.2 赔偿限额表述

| 情形 | 标准表述 |
|------|---------|
| 累计赔偿限额 | 在保险期间内，保险人对所有赔偿请求的累计赔偿金额不得超过本保险合同载明的累计赔偿限额 |
| 每次事故限额 | 对每次事故的赔偿金额不得超过本保险合同载明的每次事故赔偿限额 |
| 每人赔偿限额 | 对每个受害人的赔偿金额不得超过本保险合同载明的每人赔偿限额 |
| 财产损失限额 | 对每次事故中第三者财产损失的赔偿金额不得超过本保险合同载明的每次事故财产损失赔偿限额 |

### 1.3 责任免除表述

**通用免责：**
| 免责类型 | 标准表述 |
|---------|---------|
| 战争暴动 | 战争、敌对行动、军事行为、武装冲突、罢工、暴乱、恐怖活动 |
| 核风险 | 核辐射、核爆炸、核污染及其他放射性污染 |
| 不可抗力 | 自然灾害：地震、火山爆发、海啸等不可抗力 |
| 故意行为 | 被保险人的故意行为 |
| 违法犯罪 | 被保险人从事违法犯罪活动 |
| 精神损失 | 精神损害赔偿 |
| 间接损失 | 间接损失、罚款、罚金及惩罚性赔偿 |

**财产险专项免责：**
| 免责类型 | 标准表述 |
|---------|---------|
| 自然损耗 | 保险标的的自然损耗、自然磨损、自然蒸发 |
| 内在缺陷 | 保险标的的内在缺陷、霉变、氧化 |
| 保管不当 | 被保险人未按照国家有关消防、安全使用规范管理保险标的 |
| 贬值 | 保险标的的市场价值贬落 |

**责任险专项免责：**
| 免责类型 | 标准表述 |
|---------|---------|
| 工伤保险 | 被保险人的雇员因其工伤、职业病等依据《工伤保险条例》应由工伤保险基金支付的费用 |
| 已有疾病 | 被保险人雇员因疾病、分娩、流产及因此产生的相关费用 |
| 行政处罚 | 罚款、罚金及惩罚性赔偿 |
| 合同责任 | 因合同的约定而承担的合同责任，除非该责任依法应由被保险人承担 |

### 1.4 赔偿处理表述

**索赔材料：**
| 材料类型 | 标准表述 |
|---------|---------|
| 基础材料 | 保险单正本、索赔申请、损失清单、费用发票 |
| 事故证明 | 事故证明材料：包括但不限于警方报告、消防证明、气象证明等 |
| 身份证明 | 被保险人及受益人的身份证明 |
| 医疗材料 | 医院诊断证明、病历资料、医疗费用清单、伤残鉴定报告 |
| 死亡材料 | 死亡证明、户口注销证明、受益人身份证明 |

**时限标准：**
| 环节 | 时限 | 标准表述 |
|------|------|---------|
| 立案受理 | 3 工作日 | 收到索赔申请后 3 个工作日内完成立案受理 |
| 资料补正 | 一次性通知 | 收到完整材料后一次性通知需补充的材料 |
| 核赔确认 | 30 日 | 收到完整索赔材料后 30 日内完成核定 |
| 支付赔款 | 10 日 | 达成赔偿协议后 10 日内支付赔款 |
| 拒赔通知 | 3 日 | 做出核定之日起 3 日内发出拒赔通知书并说明理由 |

### 1.5 费率与保费表述

| 项目 | 标准表述 |
|------|---------|
| 费率表 | 详见本保险合同附带的费率表 |
| 保费计算 | 保费 = 基准保费 × 费率调整系数 |
| 费率因子 | 调整系数由以下因子综合确定：风险等级、行业类别、投保规模、历史赔付记录等 |
| 续期保费 | 应按约定及时支付续期保费，逾期未支付的，可给予 30 日宽限期 |

## 四、Word 文档格式规范（详细）

### 格式要求

| 元素 | 字体/字号 | 对齐 | 缩进 | 行距 | 段前 | 段后 |
|------|-----------|------|------|------|------|------|
| 文档标题 | 宋体二号（22pt）加粗 | 居中 | — | 固定值28磅 | 0 | 0 |
| 大纲1级（章节标题）| 仿宋三号（16pt）加粗 | 居中 | — | 固定值28磅 | 0 | 0 |
| 大纲2级（（一）（二）等）| 仿宋三号加粗 | 两端对齐 | 首行缩进2字符 | 固定值28磅 | 0 | 0 |
| 正文编号（第X条等）| 仿宋三号加粗（编号）+不加粗（内容） | 两端对齐 | 首行缩进2字符 | 固定值28磅 | 0 | 0 |
| 正文内容 | 仿宋三号不加粗 | 两端对齐 | 首行缩进2字符 | 固定值28磅 | 0 | 0 |
| 子项编号（（一）（二）等）| 仿宋三号不加粗 | 两端对齐 | 首行缩进4字符 | 固定值28磅 | 0 | 0 |
| 表格 | 表头加粗居中12pt，蓝色底色 | — | — | — | — | — |
| 页边距 | 上下2.54cm，左右3.17cm（A4纸） | — | — | — | — | — |

### 关键格式说明

1. **行距必须选"固定值"28磅**，不是28倍、不是最小值、不是多倍行距
2. **所有元素（标题、正文、编号、子项）的段前间距 = 0行，段后间距 = 0行**
3. **首行缩进用字符单位**：正文2字符，子项4字符（1字符 ≈ 210 twips，三号字时约420 twips）
4. **页面设置**：A4纸张，上下2.54cm，左右3.17cm

### Python 实现要点

```python
from docx import Document
from docx.shared import Pt, Cm, Twips
from docx.enum.text import WD_ALIGN_PARAGRAPH, WD_LINE_SPACING

# === 核心常量 ===

FONT_TITLE_SIZE = Pt(22)   # 二号
FONT_BODY_SIZE = Pt(16)    # 三号
LINE_SPACING = Pt(28)      # 固定值28磅（python-docx 直接传 Pt 对象）
INDENT_2CHAR = 2 * 420     # 2字符缩进 = 840 twips
INDENT_4CHAR = 4 * 420     # 4字符缩进 = 1680 twips

# === 每个段落的行距和间距必须这样设置 ===
p.paragraph_format.space_before = Pt(0)   # 段前 0
p.paragraph_format.space_after = Pt(0)    # 段后 0
p.paragraph_format.line_spacing_rule = WD_LINE_SPACING.EXACTLY  # 固定值
p.paragraph_format.line_spacing = Pt(28)  # 28磅

# === 注意事项 ===
# 1. 中文引号注意：JS 中用英文单引号，输出时再转中文引号
# 2. 表格表头加粗居中，蓝色底色用 RGBColor(0xD6, 0xE4, 0xF0)
# 3. 不要给任何段落设置非零的 space_before / space_after
```

## 五、常见陷阱与反模式（详细版）

> 借鉴 Insurance-Skills 质量体系，整理保险产品开发高频错误。

### 5.1 条款撰写常见陷阱

| # | 反模式 | 严重性 | 正确做法 |
|---|--------|--------|----------|
| 1 | 责任范围与除外责任重叠 | 🔴 高 | 使用"责任边界检测"逐条核对 |
| 2 | 赔偿限额表述歧义 | 🔴 高 | 明确区分"每次事故限额"vs"累计限额"vs"每人限额" |
| 3 | 费率低于精算基础 | 🔴 高 | 必须经过费率测算，不可凭经验定价 |
| 4 | 缺少监管依据引用 | 🟡 中 | 每条责任须附法规依据 |
| 5 | 免责条款未显著提示 | 🟡 中 | 加粗或颜色标注 |
| 6 | 条款章节缺失 | 🟡 中 | 必须包含13章节标准结构 |

### 5.2 典型错误示例

**❌ 错误1：责任与免责矛盾**
```markdown
# 错误写法
保险人对因火灾造成的损失负责赔偿，但以下情况除外：
（一）因地震造成的损失；
（二）保险标的因火灾造成的间接损失。
```
✅ **正确写法**：删除矛盾表述，明确地震为除外责任后，火灾责任不含地震原因。

**❌ 错误2：限额表述不清**
```markdown
# 错误写法
保险人对每次事故的赔偿金额不超过100万元。
```
✅ **正确写法**：
```markdown
第六条 赔偿限额
（一）累计赔偿限额：人民币[X]万元
（二）每次事故赔偿限额：人民币[X]万元
（三）每次事故每人赔偿限额：人民币[X]万元
```

**❌ 错误3：缺少费率测算**
```markdown
# 错误写法
保费按照行业惯例确定为千分之一。
```
✅ **正确写法**：
```markdown
保费 = 基准保费 × 费率调整系数
基准保费依据近三年行业赔付数据精算确定
费率调整系数 = 风险等级系数 × 历史赔付系数 × 规模系数
```

### 5.3 费率厘定常见陷阱

| # | 反模式 | 严重性 | 正确做法 |
|---|--------|--------|----------|
| 1 | 照搬竞品费率 | 🔴 高 | 根据本公司历史数据调整 |
| 2 | 忽视逆选择风险 | 🔴 高 | 设置免赔额/限额控制高风险 |
| 3 | 基准费率不含风险边际 | 🟡 中 | 包含利润边际和应急费用 |
| 4 | 未考虑费用加载 | 🟡 中 | 包含获取费、管理费、间接费 |

### 5.4 可行性报告常见陷阱

| # | 反模式 | 严重性 | 正确做法 |
|---|--------|--------|----------|
| 1 | 缺少监管依据引用 | 🔴 高 | 必须三段式引用 |
| 2 | 操作风险流于形式 | 🟡 中 | 详细说明6方面风控措施 |
| 3 | 市场分析数据陈旧 | 🟡 中 | 使用最新行业数据 |
