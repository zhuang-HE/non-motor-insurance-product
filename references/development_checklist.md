## 步骤1：确定保险标的概念

### 1.1 标的合法性检查
- [ ] 标的是否具有法律认可的财产或利益属性
- [ ] 是否属于不可保范围（土地、矿藏、货币票证、违章建筑、非法财产等）
- [ ] 是否有明确的权属关系

### 1.2 可保五要素验证
| 要素 | 检查要点 |
|------|---------|
| 合法性 | 标的不违反法律法规 |
| 确定性 | 标的范围可明确界定 |
| 可估价性 | 可用货币衡量价值 |
| 可转移性 | 风险可通过保险机制转移 |
| 偶然性 | 损失具有偶然性，非必然发生 |

### 1.3 标的分类与估价方法
| 标的类型 | 常见实例 | 估价方法 |
|---------|---------|---------|
| 有形财产 | 设备、存货、建筑物 | 重置价值/账面价值/市场价值 |
| 无形财产 | 利润、租金 | 预期收益/合同金额 |
| 法律责任 | 侵权赔偿、合同赔偿 | 赔偿限额 |
| 合同权益 | 租赁权、抵押权 | 合同价值/权益价值 |

### 1.4 科技保险特殊标的处理
科技保险标的与传统财产保险有本质区别，通常为"费用"或"知识产权"而非有形财产：

| 科技险标的类型 | 实例 | 估价方式 | 风险特点 |
|--------------|------|---------|---------|
| 转化/研发费用 | 中试项目费用、成果转化费用 | 预算金额/合同金额 | 失败概率高，道德风险需管控 |
| 知识产权 | 专利权、软件著作权 | 评估价值/维权成本 | 价值评估难，侵权取证难 |
| 科研设备 | 共享科研仪器 | 重置价值/实际价值 | 使用频率影响风险 |
| 技术成果 | 专有技术、技术方案 | 转让价值/许可费 | 泄密风险高，保险利益认定复杂 |

### 1.6 新兴产业标的处理
- **网络安全标的**：数据资产、网络系统安全义务——估价依赖数据分类分级结果
- **绿色保险标的**：碳汇（需第三方核证）、新能源设施、ESG管理义务
- **低空经济标的**：无人机/eVTOL——需区分消费级/工业级/载人级
- **数据资产标的**：数据集合——估价依赖数据质量×应用场景×合规程度
- **生物医药标的**：临床试验项目、药品上市后风险——需GCP/GMP资质审查

### 1.7 特殊标的处理
- **特约可保标的**：金银珠宝、古董、便携设备等需特别约定并载明价值
- **高风险标的**：危化品、烟花爆竹、大型活动等需特殊核保流程
- **复合标的**：一个产品包含多种标的需要分项列明

## 步骤2：明确投保人、被保险人身份

### 2.1 投保人资格审查
- [ ] 具有完全民事行为能力
- [ ] 对保险标的具有保险利益
- [ ] 信用状况良好
- [ ] 如为企业：营业执照有效、经营范围合法

### 2.2 保险利益类型判定
| 利益类型 | 典型场景 |
|---------|---------|
| 所有权 | 标的属于被保险人 |
| 使用权 | 租赁、借用关系 |
| 保管权 | 仓储、物流关系 |
| 抵押权 | 贷款抵押关系 |
| 合同权益 | 基于合同产生的利益 |
| 法定责任 | 雇主责任、产品责任 |

### 2.3 关键确认事项
- 投保人与被保险人是否为同一主体（常见：是）
- 如非同一主体：关系说明 + 利益证明文件
- 受益人：通常为被保险人本人（财产险/责任险不指定受益人）

### 2.4 科技保险投保人特殊审查
科技保险的投保人/被保险人通常为高校、科研机构、科技企业，需额外审查：

| 审查项 | 审查内容 | 数据来源（待数据库对接） |
|--------|---------|----------------------|
| 资质认定 | 是否为高新技术企业/科技型中小企业 | 科技部政务服务平台 |
| 项目认定 | 项目是否经政府部门立项/备案/认定 | 地方科技局数据 |
| 专利状态 | 专利有效性、权属清晰度 | 国家知识产权局 |
| 资金来源 | 是否获得政府财政资金支持（影响赔付计算） | 企业财务数据 |
| 信用状况 | 企业信用评级、行政处罚记录 | 征信系统/企查查 |

### 2.5 网络安全保险投保人特殊审查

| 审查项 | 审查内容 | 数据来源 |
|--------|---------|---------|
| 等保备案 | 是否完成等保2.0备案及定级（二级以上优先承保） | 公安部等保备案系统 |
| 安全基线 | 防火墙、加密、访问控制、日志审计等是否部署 | 安全评估报告 |
| 历史安全事件 | 过去3年是否有数据泄露/网络攻击事件 | 企业安全记录 |
| 数据分类分级 | 是否完成数据资产盘点和分类分级 | 数据治理报告 |
| 合规认证 | ISO 27001、SOC 2等安全认证 | 认证机构 |

### 2.6 绿色保险投保人特殊审查

| 审查项 | 审查内容 | 数据来源 |
|--------|---------|---------|
| 绿色认证 | 是否获得绿色建筑/绿色项目认证 | 住建部/发改委 |
| 碳排放数据 | 碳排放量、减排目标 | 企业ESG报告 |
| 环保合规 | 环评批复、排污许可证是否有效 | 生态环境部 |
| ESG评级 | 企业ESG评级情况 | MSCI/Wind/商道融绿 |

## 步骤3：初步明确保险责任范围

### 3.1 产品类型选择
| 类型 | 保障范围 | 适用场景 |
|------|---------|---------|
| 基本险 | 列明风险（火灾、爆炸、雷击） | 成本敏感的基础保障 |
| 综合险 | 扩展自然灾害 | 需要自然灾害保障 |
| 一切险 | 除外列明（不保的列出来，其余都保） | 高价值、全面保障 |
| 特定风险险 | 针对特定风险场景 | 科技险、知识产权险等创新险种 |

### 3.2 责任设计要点
- **主险责任**：明确承保原因（自然灾害/意外事故/系统故障/认定失败等）
- **附加险**（可选）：盗窃扩展、清理费用、专业费用、灭火费用等
- **除外责任**：通用除外（战争、核、故意行为、行政司法）+ 产品特定除外

### 3.2.1 财产险类责任设计模式
- **基本险**：火灾 + 爆炸 + 雷击 + 飞行物体坠落
- **综合险**：基本险 + 暴雨、洪水、台风、暴风、龙卷风、雪灾、雹灾、冰凌、泥石流等自然灾害
- **一切险**：除外列明制（除责任免除外的所有自然灾害和意外事故）
- **机器损坏险**：设计错误 + 操作失误 + 电气短路 + 离心断裂 + 超负荷等

### 3.2.2 科技/创新险责任设计模式（与传统财产险完全不同）
| 险种类型 | 责任触发条件 | 赔偿对象 | 参考案例 |
|---------|------------|---------|---------|
| 费用损失险 | 项目经认定**失败或未能达到预期成果而终止** | 已投入的费用（材料/人工/设备/评审等） | 科技成果转化险、中试险 |
| 知识产权维权险 | 专利被第三方侵权且已立案受理 | 调查费 + 法律费用 | 专利执行险 |
| 设备损坏险 | 科研仪器因特定原因损坏 | 修理/重置费用 | 科研仪器共享险 |
| 技术责任险 | 系统故障致第三方人身/财产损失 | 依法应承担的赔偿责任 | 自动控制系统责任险 |

**关键区别**：科技险的触发条件通常是"认定失败"或"侵权行为"，而非自然灾害或意外事故。

### 3.2.3 网络安全险责任设计模式

| 险种类型 | 责任触发条件 | 赔偿对象 | 行业标准 |
|---------|------------|---------|---------|
| 网络安全第一方损失险 | 安全事件（数据泄露/勒索/系统中断）发生 | 数据恢复+应急响应+营业中断+勒索赎金 | JR/T 0290-2023 |
| 网络安全责任险 | 因安全事件导致第三方数据泄露/隐私侵权 | 第三方索赔+法律费用 | JR/T 0290-2023 |
| IT错误与遗漏险 | IT服务/产品存在缺陷导致客户损失 | 客户直接损失+间接损失 | 无专项标准 |

### 3.2.4 绿色保险责任设计模式

| 险种类型 | 责任触发条件 | 赔偿对象 | 特殊考量 |
|---------|------------|---------|---------|
| 碳汇保险 | 自然灾害/意外事故致碳汇损失 | 碳汇价值减损 | 需第三方碳汇量核证 |
| 环境污染责任险 | 突发环境污染事件 | 第三方人身/财产损害+清理费用 | 绿色版需覆盖渐进性污染 |
| 新能源电站综合险 | 自然灾害/意外事故致电站损失 | 设备修复/重置+营业中断 | 需考虑设备衰减 |

### 3.2.5 低空经济保险责任设计模式

| 险种类型 | 责任触发条件 | 赔偿对象 | 特殊考量 |
|---------|------------|---------|---------|
| 无人机机身一切险 | 飞行/滑行/地面任何意外原因 | 机身损失或损坏+施救费用 | 电路/电池/进水通常除外 |
| 无人机第三者责任险 | 无人机致第三者人身/财产损失 | 依法承担的赔偿责任+法律费用 | 操作员自身损害除外 |
| eVTOL综合险 | 载人飞行中的意外事故 | 机身+第三者+乘客责任 | 监管适航认证是承保前提 |

### 3.3 除外责任设计原则
**通用除外（不可删减）**：
1. 投保人/被保险人故意或重大过失
2. 战争、军事行动、恐怖活动
3. 核辐射、核污染
4. 行政行为或司法行为
5. 保险标的内在缺陷、自然磨损

**产品特定除外（需针对性设计）**：
- 市场风险（价格波动、经济性不达预期）—— 费用损失险必须除外
- 网络攻击（除非特别承保）—— 自动控制系统险除外
- 知识产权侵犯（除非特别承保）—— 技术责任险除外
- 违规操作/超限使用/违反操作规程 —— 技术设备类除外
- 投保前已知缺陷或已提起诉讼的纠纷
- 非被保险人自主发明且独家实施的专利（专利执行险核心除外）

### 3.4 除外责任设计矩阵（按险种类型）

> ✅ = 必须除外；- = 不适用；视情况 = 根据产品定位决定
> 每个除外项均附带标准条款表述模板，可直接写入第4章（责任免除）

#### 通用除外条款（所有险种必须包含，可直接复制）

| 序号 | 除外项 | 标准条款表述 |
|------|--------|------------|
| A1 | 故意/重大过失 | "投保人或被保险人的故意行为或重大过失造成的损失、费用和责任，保险人不负责赔偿。" |
| A2 | 战争/军事/恐怖 | "因战争、军事行动、武装冲突、恐怖活动、暴乱等原因造成的损失、费用和责任，保险人不负责赔偿。" |
| A3 | 核辐射/污染 | "因核辐射、核爆炸、核污染及其他放射性污染造成的损失、费用和责任，保险人不负责赔偿。" |
| A4 | 行政/司法行为 | "因行政行为或司法行为造成的损失、费用和责任，保险人不负责赔偿。" |

| 违规操作 | - | ✅ | - | - | ✅ | **✅安全疏忽** | **✅违规处理** | **✅违规排放** | "因被保险人[违反操作规程/未按安全规范操作/违规处理数据/违规排放]造成的损失，保险人不负责赔偿。" |
| 监管处罚/罚款 | - | **✅必除** | - | - | - | **✅必除** | **✅必除** | **✅必除** | "被保险人因违反法律法规被处以的行政罚款、刑事罚金及惩罚性赔偿，保险人不负责赔偿。" |
| 渐进性污染 | - | 视情况 | - | - | - | - | - | 视情况 | "因渐进性污染、自然渐变、自燃等逐渐发生的损害造成的损失，保险人不负责赔偿。" |
| 精神损害 | - | **✅必除** | - | - | - | - | - | - | "精神损害赔偿、惩罚性赔偿，保险人不负责赔偿。" |
| 间接损失 | 视情况 | **✅必除** | - | - | - | 视情况 | 视情况 | 视情况 | "因保险事故造成的间接损失、衍生损失、预期利润损失，保险人不负责赔偿。" |

#### 除外责任条款生成规则

```
1. 所有险种：必须包含 A1-A4 四项通用除外条款
2. 财产险：A1-A4 + 自然磨损内在缺陷 + 间接损失（选）
3. 责任险：A1-A4 + 投保前已知问题 + 知识产权侵权 + 监管处罚/罚款 + 精神损害 + 间接损失

5. 专利维权险：A1-A4 + 投保前已知问题（需约定追溯期）

## 步骤5：撰写保险产品条款

### 5.1 条款结构（13章节 + 2附录，必须遵循）
1. **总则** - 合同组成、投被保人定义
2. **保险标的** - 可保标的、特约可保标的、不可保标的
3. **保险责任** - 主险责任条款
4. **责任免除** - 原因免除 + 损失/费用免除
5. **保险价值/保险金额/责任限额/免赔额（率）**
6. **保险期间**
7. **保险人义务** - 条款说明、签发保单、核定赔偿
8. **投保人/被保险人义务** - 如实告知、安全维护、出险通知、单证提供
9. **赔偿处理** - 赔偿方式、计算方法、代位求偿、诉讼时效
10. **争议处理和法律适用**
11. **其他事项** - 合同成立/解除、退保规则
12. **释义** - 专业术语定义
13. **短期费率表**

### 5.2 通用条款文本（可直接引用，按需微调）

以下章节具有高度标准化特征，可直接参考案例产品中的表述微调使用：
- 总则（第1-4条）
- 保险人义务（约6条）
- 投保人被保险人义务（约7条，含如实告知、安全维护、标的转让、危险增加通知、出险通知、单证提供）
- 赔偿处理（约8条，含赔偿方式、比例计算、施救费用、代位求偿、诉讼时效）
- 争议处理（2条）
- 其他事项（4条）
- 短期费率表（固定格式）

### 5.3 需针对性撰写的章节
- **保险标的**（第2章）：根据具体险种定义标的范围
- **保险责任**（第3章）：核心差异化内容
- **责任免除**（第4章）：通用除外 + 产品特定除外
- **保险价值/金额**（第5章）：估价方式、金额确定规则
- **释义**（第12章）：产品特定术语

### 5.3.1 科技保险条款特殊注意事项
科技保险条款与传统财产险/责任险有显著差异，需注意：

| 章节 | 传统财险 | 科技保险（费用损失险/专利险等） |
|------|---------|----------------------------|
| 保险标的 | 有形财产（设备/建筑/存货） | 无形资产（费用/专利/技术成果） |
| 保险价值 | 重置价值/账面价值/市场价值 | 合同金额/预算金额/评估价值 |
| 保险期间 | 固定期限（1年） | **可能与项目周期挂钩**（如中试险到项目终止日） |
| 事故认定 | 灾害/事故发生 | **需第三方或主管部门认定**（项目失败/侵权成立） |
| 赔偿处理 | 修复/重置/现金赔偿 | 费用清单审核 + 政府资金扣除 |
| 争议处理 | 财产评估争议 | 技术鉴定争议（需约定鉴定机构） |

### 5.3.2 网络安全保险条款特殊注意事项

| 章节 | 传统财险 | 网络安全保险 |
|------|---------|------------|
| 保险标的 | 有形财产 | 网络系统/数据资产/安全义务 |
| 保险价值 | 重置/市场价值 | 数据恢复成本+应急响应费用+营业中断损失 |
| 事故认定 | 灾害/事故发生 | **安全事件确认**（需第三方取证或等保测评） |
| 保险期间 | 固定期限 | 可按事件发生制（期内索赔制） |
| 赔偿处理 | 修复/重置 | 应急响应费用预付+数据恢复+营业中断按天计算 |
| 免赔 | 固定金额/比例 | 通常为免赔天数（如7天等待期） |

### 5.3.3 绿色保险条款特殊注意事项

| 章节 | 传统财险 | 绿色保险 |
|------|---------|---------|
| 保险标的 | 有形财产 | 碳汇/绿色设施/ESG管理义务 |
| 保险价值 | 市场/重置价值 | 碳汇量×碳价（需第三方核证） |
| 事故认定 | 灾害/事故 | 碳汇损失需第三方碳汇计量验证 |
| 赔偿处理 | 直接赔偿 | 碳汇价值按事故前后碳汇量差额×碳价计算 |
| 除外责任 | 标准除外 | 渐进性污染是否除外需特别约定 |

### 5.4 条款用语规范（司法实践向）

> **核心原则**：用语精确有据、除外清晰有序、义务边界明确、争议解决可预期。语义清晰的条款在法院/仲裁中能更好支持保险公司的抗辩立场。

**关键表述快查**：
- 核保权限："分级授权、从人授信、一人一码、严禁转用"
- 保险利益：被保险人在保险事故发生时对保险标的应具有保险利益
- 诉讼时效：二年，自**知道或应当知道**保险事故发生之日起计算
- 代位求偿：自赔偿之日起，在赔偿金额范围内**依法**代位行使

#### 5.4.1 除外条款用语规范（司法最常援引的抗辩依据）

> ⚠️《保险法》第17条：保险公司对免责条款须履行**明确说明义务**；未履行说明义务的免责条款不产生效力。

| 注意事项 | 反例（存在风险） | 正例（司法友好） |
|---------|---------------|----------------|
| 具体列举，避免概括 | "一切因被保险人过失造成的损失" | "投保人或被保险人的**故意行为或重大过失**造成的损失" |
| 除外项必须有清晰边界 | "超出正常使用范围造成的损失" | "被保险人**违反制造商操作手册规定的操作规程**造成的损失" |
| 因果关系表述须明确 | "与核风险有关的损失" | "**直接或间接**由核辐射、核爆炸、核污染造成的损失" |
| 加粗/显著标识义务 | 免责条款与正文字体相同 | 免责条款**加粗**（或变色），投保单上另行提示并留存签字确认 |
| 避免"等"字兜底 | "火灾、爆炸、雷击等原因" | 明确列举所有承保原因，不使用开放性"等" |

#### 5.4.2 保险责任用语规范

| 注意事项 | 具体要求 |
|---------|---------|
| 责任触发条件精确 | 不得使用"严重损失""重大事故"等缺乏量化标准的表述；应规定具体标准（如"直接经济损失超过X万元"） |
| "意外事故"须有定义 | 在释义章节明确"意外事故"的含义（外来的、突发的、非预期的），防止争议 |
| 科技险"失败认定"须明确主体 | 应明确"由[具体政府部门/认定机构]出具认定报告方视为保险事故"，避免各执一词 |
| 赔偿对象须穷尽列举 | 可赔的费用类型（修复费/重置费/调查费/法律费等）逐条列明，不得笼统表述 |
| 责任制度须明确 | 在第一章或保险期间章节明确采用"事故发生制"还是"索赔发生制" |

#### 5.4.3 义务条款用语规范（有利于拒赔/减赔抗辩）

| 义务类型 | 用语建议 | 司法效果 |
|---------|---------|---------|
| 如实告知义务 | 明确"投保时对保险人书面询问的事项须如实回答"，并规定未如实告知的法律后果 | 支持以违反告知义务为由解除合同 |
| 安全维护义务 | 具体列明"应维持的安全措施"（如"配备防火设施并保持有效""定期安全检查"），而非笼统要求 | 被保险人未履行具体义务时可援引减赔 |
| 危险增加通知义务 | 规定"保险期间内，保险标的危险程度增加时，被保险人须在知道后**X日内**书面通知保险人" | 时限明确有助于主张未通知的免赔依据 |
| 出险通知时限 | 规定具体天数（如"**5个工作日内**书面通知"），不得仅写"及时通知" | 超期通知可依条款主张减损 |
| 施救义务 | "被保险人有责任采取一切**必要、合理**的措施以防止损失扩大" | "合理"限定可防止法院认定格式条款不公平 |

#### 5.4.4 赔偿处理条款用语规范

| 条款 | 用语建议 | 注意事项 |
|------|---------|---------|
| 诉讼时效 | "自知道或**应当**知道保险事故发生之日起**二年**内" | "应当知道"可防止被保险人以"不知道"为由主张时效中断 |
| 代位求偿 | "自赔偿之日起，在赔偿金额范围内**依法**代位行使被保险人对第三方的索赔权利" | "依法"限定防止被质疑超出法律授权 |
| 比例赔偿 | "保险金额低于保险价值时，按保险金额与保险价值之比例计算赔偿" | 表述须清晰，防止"比例不足额"产生争议 |
| 重复保险 | "存在重复保险时，本公司仅按本保单赔偿限额占所有保单限额之比例赔偿" | 防止超额赔付 |
| 施救费用 | 明确施救费用纳入赔偿范围，但"以不超过保险金额为限" | 无上限条款将导致超额赔偿（司法败诉高频漏洞） |

#### 5.4.5 释义章节用语规范（防止词义不利解释）

| 注意事项 | 具体要求 |
|---------|---------|
| 关键术语必须定义 | "意外事故""保险事故""损失""第三者"等核心词汇必须在释义中定义，不留解释空间 |
| 定义须与正文一致 | 释义中定义的术语，在条款正文中必须原词使用（不能用近义词替换） |
| 限制性定义优先 | 对承保范围内的关键术语，定义应清晰边界，防止被解释扩大 |
| "第三者"须排除内部人员 | "第三者"应明确"不包括投保人、被保险人及其雇员"，防止责任险内部索赔 |
| "重大过失"需界定 | 在释义中明确"重大过失"的认定标准，防止法院自由裁量不利于保险公司 |

#### 5.4.6 常见合规禁用表述清单

| ❌ 禁用表述 | 问题 | ✅ 建议替换 |
|-----------|------|-----------|
| "保险公司不承担任何责任" | 过于绝对，违反监管要求 | 明确列举具体免责情形 |
| "一切损失均由投保人承担" | 过于宽泛，不符合条款设计规范 | 列明具体损失类型 |
| "保险公司有权随时修改条款" | 单方变更权，无效条款 | "经双方协商一致，可变更合同条款" |
| "严重损失""重大事故" | 缺乏量化标准，产生争议 | 规定具体金额或百分比阈值 |
| "及时通知""尽快报告" | 时限不明确 | "在知道后X个工作日内书面通知" |
| "相关损失""其他损失" | 兜底性过宽 | 穷尽列举具体损失类型 |
| "一般不可抗力" | 无法律意义的模糊限定 | 直接援引《民法典》不可抗力定义 |

#### 5.4.7 其他司法实践注意事项

| 维度 | 注意事项 |
|------|---------|
| 格式条款公平性 | 避免明显加重被保险人负担的霸王条款（《民法典》第496条），否则被法院认定无效 |
| 争议解决约定 | 明确选择"仲裁"或"诉讼"，并指定仲裁机构/管辖法院（建议约定保险人所在地法院） |
| 免责条款解释规则 | 《保险法》第30条：合同约定不明时，按有利于被保险人的原则解释。应消除所有可能引发争议的模糊表述 |
| 除外条款说明义务留证 | 条款设计时预留"投保人确认已知悉免责条款"的签字栏，与条款合为一体 |

### 5.6 赔偿处理条款设计规范（防止被保险人不当得利）

> **核心原则**：保险补偿原则——被保险人因保险事故获得的赔偿，不得超过其实际遭受的损失。条款设计须构建多层防线，防止超额补偿、重复获益和道德风险。

#### 5.6.1 赔偿方式与选择权

| 赔偿方式 | 适用场景 | 撰写要点 |
|---------|---------|---------|
| 现金赔偿 | 财产损失、费用损失 | 明确计算公式（见5.6.2），防止无限制赔付 |
| 修复/恢复 | 设备损坏、财产受损 | 规定"修复至损前功能状态"，不要求"升级"；明确修复费用超过重置费用时以重置价为限 |
| 重置 | 全损或修复不经济 | 须明确"同等性能、规格的同类产品重置费用"，防止被保险人要求升级替代 |
| 替代服务 | 责任险赔付 | 费用须经保险人**事先书面同意**方可纳入赔付 |

> **撰写建议**：在条款中明确"保险人有权选择上述赔偿方式之一"——赋予保险公司赔偿方式的选择权。

#### 5.6.2 赔偿计算公式体系（防止超额赔付核心）

**财产险赔偿计算公式**：

| 情形 | 赔偿公式 | 说明 |
|------|---------|------|
| 足额投保（保险金额≥保险价值） | 赔偿金额 = 实际损失 - 免赔额 | 以实际损失为上限 |
| 不足额投保（保险金额＜保险价值） | 赔偿金额 = 实际损失 × (保险金额/保险价值) - 免赔额 | **比例赔付**，须在条款中明确写入 |
| 超额投保（保险金额＞保险价值） | 赔偿金额以保险价值为限，超出部分保费不退 | **防止超额赔付的关键** |

> ⚠️ **条款必须约定**："本保险合同的赔偿金额，以不超过保险金额为限，且以不超过保险标的的实际损失为限"——**双重上限**设定。

**责任险赔偿限额层级**：

| 限额层级 | 防止不当得利作用 |
|---------|----------------|
| 每次事故限额 | 防止单次巨额索赔 |
| 每人/每次限额 | 防止单人过度索赔 |
| 累计赔偿限额 | 防止全年累计超额 |
| 免赔额 | 强制被保险人分担，防止道德风险 |

#### 5.6.3 施救费用条款（控制附加赔付膨胀）

```
推荐标准表述：
"保险事故发生后，被保险人为防止损失扩大所支付的必要、合理的施救费用，
保险人予以赔偿，但赔偿金额以不超过本保险合同载明的保险金额为限。
施救费用与损失赔偿金额之和，不超过保险金额。"
```

| 注意事项 | 具体要求 |
|---------|---------|
| "必要且合理"限定 | 须加此限定，防止被保险人扩大施救支出 |
| 费用上限（双封顶） | 施救费用+损失赔偿之和不超过保险金额 |
| 提前告知义务 | 可约定"施救行动前须及时通知保险人"，防止事后无法核实 |

#### 5.6.4 代位求偿条款（追偿权保护）

```
推荐标准表述：
"因第三方的行为造成保险事故，保险人向被保险人赔偿后，
在赔偿金额范围内依法代位行使被保险人对第三方的索赔权利。
被保险人应当提供必要的协助。未经保险人书面同意，
被保险人不得放弃对第三方的索赔权利；否则，保险人有权拒绝赔偿或追回已付赔款。"
```

#### 5.6.5 重复保险分摊条款（防止多重赔付）

```
推荐标准表述：
"被保险人就同一保险标的、同一保险事故向多个保险人投保，
发生损失时，各保险人按各自保险金额与所有保险人保险金额总和的比例承担赔偿责任。
本公司的赔偿金额与其他保险人赔偿金额之和，不超过实际损失。"
```

#### 5.6.6 政府补偿扣除条款（科技险/绿色险必须）

```
推荐标准表述：
"保险事故发生后，被保险人已从政府财政补贴、专项扶持资金或
第三方赔偿中获得的补偿金额，应在保险赔偿金额中予以扣除。
被保险人有义务如实披露已获得或预期获得的上述补偿。"
```

| 适用险种 | 重点扣除项目 |
|---------|------------|
| 科技成果转化险/中试险 | 政府财政补贴、科技扶持资金、合作方补偿 |
| 网络安全险 | 网络安全应急处置财政拨款、责任方赔偿 |
| 碳汇保险 | 碳汇补偿基金、政府生态补偿款 |
| 环境污染责任险 | 环保基金赔付、责任方赔偿 |

#### 5.6.7 赔偿处理章节完整性检查清单（14项）

```
赔偿处理完整性检查（第9章，撰写前逐项核对）

基础框架
□ 1. 赔偿方式选择权归属明确（保险人选择权 vs 被保险人申请权）
□ 2. 赔偿金额双上限设定（不超过保险金额 + 不超过实际损失）
□ 3. 赔偿计算公式清晰（不足额投保比例赔付公式写入）

防止超额赔付
□ 4. 超额投保处理（以保险价值为限）
□ 5. 不定值保险折旧扣减条款（"实际现金价值含折旧"）
□ 6. 施救费用双封顶（施救+损失之和不超过保险金额）

防止重复得益

□ 8. 政府补偿/第三方赔偿扣除条款（科技险/绿色险必须）

#### 5.6.8 司法败诉高频漏洞对照表

| 漏洞类型 | 典型问题表述 | 法院认定 | 改进建议 |
|---------|------------|---------|---------|
| 超额赔付无上限 | 未明确"以实际损失为限"的表述 | 按保险金额全额赔付 | 加入双上限条款 |
| 未约定折旧扣减 | "按损失价值赔偿"但未说明是否含折旧 | 按新价赔付 | 明确"实际现金价值（含折旧）" |
| 代位权被抛弃 | 未约定被保险人不得放弃第三方索赔权 | 代位权丧失 | 加入禁止放弃条款 + 违反后果 |
| 施救费用无上限 | 仅约定"施救费用予以赔偿"无上限 | 独立计算叠加赔付 | 加入"与赔偿金额之和不超过保险金额" |
| 时效"应当知道"缺失 | 仅约定"自知道保险事故发生之日起" | 被保险人以不知情拖延时效 | 加入"或应当知道" |
| 重复保险未约定分摊 | 无重复保险条款 | 被保险人可向各保险人分别全额索赔 | 加入比例分摊条款 |
| 政府补偿未约定扣除 | 科技险未约定扣除政府补贴 | 被保险人同时获得补贴+保险赔付 | 加入补偿扣除条款 |

#### 5.6.9 防不当得利的法律依据速查

| 法律依据 | 条文 | 对应条款设计 |
|---------|------|------------|
| 《保险法》第12条 | 补偿原则法律基础 | 赔偿计算公式不得超过实际损失 |
| 《保险法》第55条 | 超额保险：以保险价值为限赔偿 | 超额投保处理条款 |

| 《保险法》第60条 | 代位求偿权：赔偿后依法取得代位权 | 代位求偿条款 |

### 6.1 定价方法选择

| 方法 | 适用场景 | 公式 | 数据需求 |
|------|---------|------|---------|
| **纯保费法** | 新产品无历史数据 | 毛保费 = 纯保费 / (1 - 费用率 - 利润率) | 行业损失成本、风险暴露量 |
| **损失率法** | 有经营历史数据 | 指示费率 = 当前费率 × 经验损失率 / 目标损失率 | 已发生赔款、已赚保费 |
| **Burning Cost法** | 大额/低频风险 | 基于历史赔案的层损失率 | 历史赔案数据、限额结构 |
| **类比法** | 无任何数据的新险种 | 参考类似产品费率 + 风险调整 | 同业费率参考 |

### 6.2 基础假设参数

| 参数 | 典型范围 | 说明 |
|------|---------|------|
| 预期赔付率 | 45%-65% | 新险种取高值，成熟险种取低值 |
| 费用率 | 20%-30% | 含佣金、管理费、渠道费用 |
| 利润率 | 3%-8% | 监管要求费率合理 |
| 风险附加 | 5%-15% | 不确定性补偿 |

**费率计算公式**：
```
费率 = 纯保费率 / (1 - 费用率 - 利润率)
纯保费率 = 预期损失率 × 基础损失成本
```

### 6.6 调整因子应用

**加权调整模型**：
```
因子调整系数 = ∏(1 + (等级系数 - 1) × 权重)
最终费率 = 基准费率 × 因子调整系数
```

### 6.6.1 再保险对费率的影响
- 再保成本通常占毛保费的 5%-15%
- 成数分保：分出佣金率 20%-35%
- 溢额分保：大额业务需获得再保报价后再确定最终费率
- 创新险种需先获得再保支持才能定价
- **净费率** = 毛费率 - 再保分出成本

### 6.6.2 信用保证险定价逻辑（与财产险/责任险完全不同）
信保定价核心公式：`预期损失 = PD（违约概率）× LGD（违约损失率）× EAD（风险暴露）`

| 因子 | 权重 | 说明 |
|------|------|------|
| 企业信用评级 | 35% | AAA 0.70 → D 5.00 |
| 担保方式 | 25% | 足额抵押0.6 → 无担保1.5 |
| 行业周期性 | 20% | 稳定0.7 → 高波动1.5 |
| 保险期间 | 10% | ≤1年0.8 → >3年1.3 |
| 集中度 | 10% | ≤5%敞口0.8 → >10%1.5 |

**信保特殊注意**：需加载系统性风险附加（经济下行时违约集中爆发）；逆选择风险高需逐笔审批。

### 6.7 费率合理性校验

- [ ] 预期赔付率是否在 45%-65% 范围内
- [ ] 高风险因子是否已充分反映在费率中
