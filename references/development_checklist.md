## 步骤1：确定保险标的概念

### 1.1 标的合法性检查
- [ ] 标的是否具有法律认可的财产或利益属性
- [ ] 是否属于不可保范围（土地、矿藏、货币票证、违章建筑、非法财产等）
- [ ] 是否有明确的权属关系

### 1.2 可保五要素验证
| 要素 | 检查要点 |
|------|---------|
| 合法性 | 标的不违反法律法规 |
| 确定性 | 标的范围可明确界定 |
| 可估价性 | 可用货币衡量价值 |
| 可转移性 | 风险可通过保险机制转移 |
| 偶然性 | 损失具有偶然性，非必然发生 |

### 1.3 标的分类与估价方法
| 标的类型 | 常见实例 | 估价方法 |
|---------|---------|---------|
| 有形财产 | 设备、存货、建筑物 | 重置价值/账面价值/市场价值 |
| 无形财产 | 利润、租金 | 预期收益/合同金额 |
| 法律责任 | 侵权赔偿、合同赔偿 | 赔偿限额 |
| 合同权益 | 租赁权、抵押权 | 合同价值/权益价值 |

### 1.4 科技保险特殊标的处理
科技保险标的与传统财产保险有本质区别，通常为"费用"或"知识产权"而非有形财产：

| 科技险标的类型 | 实例 | 估价方式 | 风险特点 |
|--------------|------|---------|---------|
| 转化/研发费用 | 中试项目费用、成果转化费用 | 预算金额/合同金额 | 失败概率高，道德风险需管控 |
| 知识产权 | 专利权、软件著作权 | 评估价值/维权成本 | 价值评估难，侵权取证难 |
| 科研设备 | 共享科研仪器 | 重置价值/实际价值 | 使用频率影响风险 |
| 技术成果 | 专有技术、技术方案 | 转让价值/许可费 | 泄密风险高，保险利益认定复杂 |

### 1.6 新兴产业标的处理
- **网络安全标的**：数据资产、网络系统安全义务——估价依赖数据分类分级结果
- **绿色保险标的**：碳汇（需第三方核证）、新能源设施、ESG管理义务
- **低空经济标的**：无人机/eVTOL——需区分消费级/工业级/载人级
- **数据资产标的**：数据集合——估价依赖数据质量×应用场景×合规程度
- **生物医药标的**：临床试验项目、药品上市后风险——需GCP/GMP资质审查

### 1.7 特殊标的处理
- **特约可保标的**：金银珠宝、古董、便携设备等需特别约定并载明价值
- **高风险标的**：危化品、烟花爆竹、大型活动等需特殊核保流程
- **复合标的**：一个产品包含多种标的需要分项列明

## 步骤2：明确投保人、被保险人身份

### 2.1 投保人资格审查
- [ ] 具有完全民事行为能力
- [ ] 对保险标的具有保险利益
- [ ] 信用状况良好
- [ ] 如为企业：营业执照有效、经营范围合法

### 2.2 保险利益类型判定
| 利益类型 | 典型场景 |
|---------|---------|
| 所有权 | 标的属于被保险人 |
| 使用权 | 租赁、借用关系 |
| 保管权 | 仓储、物流关系 |
| 抵押权 | 贷款抵押关系 |
| 合同权益 | 基于合同产生的利益 |
| 法定责任 | 雇主责任、产品责任 |

### 2.3 关键确认事项
- 投保人与被保险人是否为同一主体（常见：是）
- 如非同一主体：关系说明 + 利益证明文件
- 受益人：通常为被保险人本人（财产险/责任险不指定受益人）

### 2.4 科技保险投保人特殊审查
科技保险的投保人/被保险人通常为高校、科研机构、科技企业，需额外审查：

| 审查项 | 审查内容 | 数据来源（待数据库对接） |
|--------|---------|----------------------|
| 资质认定 | 是否为高新技术企业/科技型中小企业 | 科技部政务服务平台 |
| 项目认定 | 项目是否经政府部门立项/备案/认定 | 地方科技局数据 |
| 专利状态 | 专利有效性、权属清晰度 | 国家知识产权局 |
| 资金来源 | 是否获得政府财政资金支持（影响赔付计算） | 企业财务数据 |
| 信用状况 | 企业信用评级、行政处罚记录 | 征信系统/企查查 |

### 2.5 网络安全保险投保人特殊审查

| 审查项 | 审查内容 | 数据来源 |
|--------|---------|---------|
| 等保备案 | 是否完成等保2.0备案及定级（二级以上优先承保） | 公安部等保备案系统 |
| 安全基线 | 防火墙、加密、访问控制、日志审计等是否部署 | 安全评估报告 |
| 历史安全事件 | 过去3年是否有数据泄露/网络攻击事件 | 企业安全记录 |
| 数据分类分级 | 是否完成数据资产盘点和分类分级 | 数据治理报告 |
| 合规认证 | ISO 27001、SOC 2等安全认证 | 认证机构 |

### 2.6 绿色保险投保人特殊审查

| 审查项 | 审查内容 | 数据来源 |
|--------|---------|---------|
| 绿色认证 | 是否获得绿色建筑/绿色项目认证 | 住建部/发改委 |
| 碳排放数据 | 碳排放量、减排目标 | 企业ESG报告 |
| 环保合规 | 环评批复、排污许可证是否有效 | 生态环境部 |
| ESG评级 | 企业ESG评级情况 | MSCI/Wind/商道融绿 |

## 步骤3：初步明确保险责任范围

### 3.1 产品类型选择
| 类型 | 保障范围 | 适用场景 |
|------|---------|---------|
| 基本险 | 列明风险（火灾、爆炸、雷击） | 成本敏感的基础保障 |
| 综合险 | 扩展自然灾害 | 需要自然灾害保障 |
| 一切险 | 除外列明（不保的列出来，其余都保） | 高价值、全面保障 |
| 特定风险险 | 针对特定风险场景 | 科技险、知识产权险等创新险种 |

### 3.2 责任设计要点
- **主险责任**：明确承保原因（自然灾害/意外事故/系统故障/认定失败等）
- **附加险**（可选）：盗窃扩展、清理费用、专业费用、灭火费用等
- **除外责任**：通用除外（战争、核、故意行为、行政司法）+ 产品特定除外

### 3.2.1 财产险类责任设计模式
- **基本险**：火灾 + 爆炸 + 雷击 + 飞行物体坠落
- **综合险**：基本险 + 暴雨、洪水、台风、暴风、龙卷风、雪灾、雹灾、冰凌、泥石流等自然灾害
- **一切险**：除外列明制（除责任免除外的所有自然灾害和意外事故）
- **机器损坏险**：设计错误 + 操作失误 + 电气短路 + 离心断裂 + 超负荷等

### 3.2.2 科技/创新险责任设计模式（与传统财产险完全不同）
| 险种类型 | 责任触发条件 | 赔偿对象 | 参考案例 |
|---------|------------|---------|---------|
| 费用损失险 | 项目经认定**失败或未能达到预期成果而终止** | 已投入的费用（材料/人工/设备/评审等） | 科技成果转化险、中试险 |
| 知识产权维权险 | 专利被第三方侵权且已立案受理 | 调查费 + 法律费用 | 专利执行险 |
| 设备损坏险 | 科研仪器因特定原因损坏 | 修理/重置费用 | 科研仪器共享险 |
| 技术责任险 | 系统故障致第三方人身/财产损失 | 依法应承担的赔偿责任 | 自动控制系统责任险 |

**关键区别**：科技险的触发条件通常是"认定失败"或"侵权行为"，而非自然灾害或意外事故。

### 3.2.3 网络安全险责任设计模式

| 险种类型 | 责任触发条件 | 赔偿对象 | 行业标准 |
|---------|------------|---------|---------|
| 网络安全第一方损失险 | 安全事件（数据泄露/勒索/系统中断）发生 | 数据恢复+应急响应+营业中断+勒索赎金 | JR/T 0290-2023 |
| 网络安全责任险 | 因安全事件导致第三方数据泄露/隐私侵权 | 第三方索赔+法律费用 | JR/T 0290-2023 |
| IT错误与遗漏险 | IT服务/产品存在缺陷导致客户损失 | 客户直接损失+间接损失 | 无专项标准 |

### 3.2.4 绿色保险责任设计模式

| 险种类型 | 责任触发条件 | 赔偿对象 | 特殊考量 |
|---------|------------|---------|---------|
| 碳汇保险 | 自然灾害/意外事故致碳汇损失 | 碳汇价值减损 | 需第三方碳汇量核证 |
| 环境污染责任险 | 突发环境污染事件 | 第三方人身/财产损害+清理费用 | 绿色版需覆盖渐进性污染 |
| 新能源电站综合险 | 自然灾害/意外事故致电站损失 | 设备修复/重置+营业中断 | 需考虑设备衰减 |

### 3.2.5 低空经济保险责任设计模式

| 险种类型 | 责任触发条件 | 赔偿对象 | 特殊考量 |
|---------|------------|---------|---------|
| 无人机机身一切险 | 飞行/滑行/地面任何意外原因 | 机身损失或损坏+施救费用 | 电路/电池/进水通常除外 |
| 无人机第三者责任险 | 无人机致第三者人身/财产损失 | 依法承担的赔偿责任+法律费用 | 操作员自身损害除外 |
| eVTOL综合险 | 载人飞行中的意外事故 | 机身+第三者+乘客责任 | 监管适航认证是承保前提 |

### 3.3 除外责任设计原则
**通用除外（不可删减）**：
1. 投保人/被保险人故意或重大过失
2. 战争、军事行动、恐怖活动
3. 核辐射、核污染
4. 行政行为或司法行为
5. 保险标的内在缺陷、自然磨损

**产品特定除外（需针对性设计）**：
- 市场风险（价格波动、经济性不达预期）—— 费用损失险必须除外
- 网络攻击（除非特别承保）—— 自动控制系统险除外
- 知识产权侵犯（除非特别承保）—— 技术责任险除外
- 违规操作/超限使用/违反操作规程 —— 技术设备类除外
- 投保前已知缺陷或已提起诉讼的纠纷
- 非被保险人自主发明且独家实施的专利（专利执行险核心除外）

### 3.4 除外责任设计矩阵（按险种类型）

> ✅ = 必须除外；- = 不适用；视情况 = 根据产品定位决定
> 每个除外项均附带标准条款表述模板，可直接写入第4章（责任免除）

#### 通用除外条款（所有险种必须包含，可直接复制）

| 序号 | 除外项 | 标准条款表述 |
|------|--------|------------|
| A1 | 故意/重大过失 | "投保人或被保险人的故意行为或重大过失造成的损失、费用和责任，保险人不负责赔偿。" |
| A2 | 战争/军事/恐怖 | "因战争、军事行动、武装冲突、恐怖活动、暴乱等原因造成的损失、费用和责任，保险人不负责赔偿。" |
| A3 | 核辐射/污染 | "因核辐射、核爆炸、核污染及其他放射性污染造成的损失、费用和责任，保险人不负责赔偿。" |
| A4 | 行政/司法行为 | "因行政行为或司法行为造成的损失、费用和责任，保险人不负责赔偿。" |

### 5.2 通用条款文本（可直接引用，按需微调）

以下章节具有高度标准化特征，可直接参考案例产品中的表述微调使用：

### 5.3 需针对性撰写的章节

- **释义**（第12章）：产品特定术语

### 5.3.1 科技保险条款特殊注意事项
科技保险条款与传统财产险/责任险有显著差异，需注意：

| 章节 | 传统财险 | 科技保险（费用损失险/专利险等） |
|------|---------|----------------------------|
| 保险标的 | 有形财产（设备/建筑/存货） | 无形资产（费用/专利/技术成果） |
| 保险价值 | 重置价值/账面价值/市场价值 | 合同金额/预算金额/评估价值 |
| 保险期间 | 固定期限（1年） | **可能与项目周期挂钩**（如中试险到项目终止日） |
| 事故认定 | 灾害/事故发生 | **需第三方或主管部门认定**（项目失败/侵权成立） |
| 赔偿处理 | 修复/重置/现金赔偿 | 费用清单审核 + 政府资金扣除 |
| 争议处理 | 财产评估争议 | 技术鉴定争议（需约定鉴定机构） |

### 5.3.2 网络安全保险条款特殊注意事项

| 章节 | 传统财险 | 网络安全保险 |
|------|---------|------------|
| 保险标的 | 有形财产 | 网络系统/数据资产/安全义务 |
| 保险价值 | 重置/市场价值 | 数据恢复成本+应急响应费用+营业中断损失 |
| 事故认定 | 灾害/事故发生 | **安全事件确认**（需第三方取证或等保测评） |
| 保险期间 | 固定期限 | 可按事件发生制（期内索赔制） |
| 赔偿处理 | 修复/重置 | 应急响应费用预付+数据恢复+营业中断按天计算 |
| 免赔 | 固定金额/比例 | 通常为免赔天数（如7天等待期） |

### 5.3.3 绿色保险条款特殊注意事项

| 章节 | 传统财险 | 绿色保险 |
|------|---------|---------|
| 保险标的 | 有形财产 | 碳汇/绿色设施/ESG管理义务 |
| 保险价值 | 市场/重置价值 | 碳汇量×碳价（需第三方核证） |
| 事故认定 | 灾害/事故 | 碳汇损失需第三方碳汇计量验证 |
| 赔偿处理 | 直接赔偿 | 碳汇价值按事故前后碳汇量差额×碳价计算 |
| 除外责任 | 标准除外 | 渐进性污染是否除外需特别约定 |

### 5.4 关键表述规范
- 核保权限："分级授权、从人授信、一人一码、严禁转用"

### 6.1 定价方法选择

| 方法 | 适用场景 | 公式 | 数据需求 |
|------|---------|------|---------|
| **纯保费法** | 新产品无历史数据 | 毛保费 = 纯保费 / (1 - 费用率 - 利润率) | 行业损失成本、风险暴露量 |
| **损失率法** | 有经营历史数据 | 指示费率 = 当前费率 × 经验损失率 / 目标损失率 | 已发生赔款、已赚保费 |
| **Burning Cost法** | 大额/低频风险 | 基于历史赔案的层损失率 | 历史赔案数据、限额结构 |
| **类比法** | 无任何数据的新险种 | 参考类似产品费率 + 风险调整 | 同业费率参考 |

### 6.2 基础假设参数

| 参数 | 典型范围 | 说明 |
|------|---------|------|
| 预期赔付率 | 45%-65% | 新险种取高值，成熟险种取低值 |
| 费用率 | 20%-30% | 含佣金、管理费、渠道费用 |
| 利润率 | 3%-8% | 监管要求费率合理 |
| 风险附加 | 5%-15% | 不确定性补偿 |

**费率计算公式**：
```
费率 = 纯保费率 / (1 - 费用率 - 利润率)
纯保费率 = 预期损失率 × 基础损失成本
```

### 6.3 财产险风险因子体系

| 因子类别 | 具体因子 | 权重 | 等级示例 |
|---------|---------|------|---------|
| 标的特征 | 建筑结构 | 25% | 钢混0.8 / 砖混1.0 / 砖木1.3 |
| 防护设施 | 消防设施 | 20% | 自动喷淋0.75 / 仅灭火器1.1 / 无1.5 |

### 6.6 调整因子应用

**加权调整模型**：
```
因子调整系数 = ∏(1 + (等级系数 - 1) × 权重)
最终费率 = 基准费率 × 因子调整系数
```

### 6.6.1 再保险对费率的影响
- 再保成本通常占毛保费的 5%-15%
- 成数分保：分出佣金率 20%-35%
- 溢额分保：大额业务需获得再保报价后再确定最终费率
- 创新险种需先获得再保支持才能定价
- **净费率** = 毛费率 - 再保分出成本

### 6.6.2 信用保证险定价逻辑（与财产险/责任险完全不同）
信保定价核心公式：`预期损失 = PD（违约概率）× LGD（违约损失率）× EAD（风险暴露）`

| 因子 | 权重 | 说明 |
|------|------|------|
| 企业信用评级 | 35% | AAA 0.70 → D 5.00 |
| 担保方式 | 25% | 足额抵押0.6 → 无担保1.5 |
| 行业周期性 | 20% | 稳定0.7 → 高波动1.5 |
| 保险期间 | 10% | ≤1年0.8 → >3年1.3 |
| 集中度 | 10% | ≤5%敞口0.8 → >10%1.5 |

**信保特殊注意**：需加载系统性风险附加（经济下行时违约集中爆发）；逆选择风险高需逐笔审批。

### 6.7 费率合理性校验

- [ ] 最终费率是否在行业合理区间内（参考同业费率）
- [ ] 预期赔付率是否在 45%-65% 范围内
- [ ] 高风险因子是否已充分反映在费率中
- [ ] 是否留有再保成本空间（通常 5%-15%）
- [ ] 与公司整体综合成本率目标是否匹配
- [ ] 费率偏离行业平均水平是否超过 ±30%（金监局关注阈值）

```python
from docx import Document
from docx.shared import Pt, Inches
from docx.enum.text import WD_ALIGN_PARAGRAPH

doc = Document()
# 设置中文字体

style.font.name = '宋体'
style._element.rPr.rFonts.set(qn('w:eastAsia'), '宋体')
# ... 按需添加内容
doc.save('output.docx')
```
