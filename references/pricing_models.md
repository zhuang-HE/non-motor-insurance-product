# 非车险定价模型与精算方法

> 本文件为定价知识的**深度参考**，与 `development_checklist.md` 步骤6互补（checklist侧重操作检查项，本文件侧重方法论）。
> 最后更新：2026-03-30

---

## 一、定价基本原则

### 1.1 充足性原则
保费应足以覆盖：
- 预期赔款（纯保费）
- 理赔费用
- 承保费用（佣金+管理费+渠道费）
- 风险附加（不确定性补偿）
- 利润附加

### 1.2 合理性原则
- 费率应与风险水平匹配
- 避免交叉补贴
- 考虑市场竞争
- **不得显著偏离行业平均水平**（金监局费率监管要求，偏离度通常≤±30%）

### 1.3 公平性原则
- 同风险同费率
- 风险差异应体现在费率中
- NCD（无赔款优待）系数体现经验差异化

## 二、定价方法详解

> 基本公式和因子体系详见 `development_checklist.md` 步骤6，此处补充每种方法的操作指引和适用条件。

### 2.1 纯保费法（新产品首选）
```
纯保费 = 预期赔款 / 风险单位数
毛保费 = 纯保费 / (1 - 费用率 - 利润率)
```
**操作要点**：
- 预期赔款可来自行业损失成本数据或再保报价
- 风险单位数 = 保险金额总和 / 平均保险金额
- 费用率通常取 20%-30%
- 新产品应使用较高风险附加（10%-15%）

### 2.2 损失率法（有历史数据时使用）
```
指示费率 = 当前费率 × 经验损失率 / 目标损失率
经验损失率 = 已发生赔款 / 已赚保费
```
**操作要点**：
- 至少需要3年经验数据
- 需对数据进行趋势化和通货膨胀调整
- 大额赔案应单独处理（分离法）

### 2.3 Burning Cost 法（大额/低频风险）
**适用场景**：超额责任险、大额财产险的自留部分定价

**操作步骤**：
1. 收集历史赔案数据（≥5年）
2. 按损失金额排序，筛选进入超赔层的赔案
3. 计算层损失率 = 层内赔款总和 / 限额
4. 调整趋势化和通货膨胀
5. 除以观察年限得到年均 Burning Cost
6. 加上风险附加（100/80 或 100/70 法）

### 2.4 类比法（无数据的新险种）
**操作步骤**：
1. 识别最相似的可比产品
2. 分析风险差异（标的类型、风险频率、损失严重度）
3. 确定调整系数（经验判断+专家意见）
4. 最终费率 = 可比产品费率 × 调整系数

**适用**：科技保险、网络安全保险、未来产业保险等新产品

## 三、再保险对定价的影响

> ⚠️ 这是 checklist 步骤6未覆盖的重要内容。实务中，再保成本直接影响最终费率。

### 3.1 再保险方式对定价的影响

| 再保方式 | 影响方式 | 典型成本 |
|---------|---------|---------|
| **成数分保** | 按比例分出保费和赔款，影响净保费收入 | 分出佣金率 20%-35% |
| **溢额分保** | 超过自留额的部分分出，大额业务影响大 | 分出佣金率 25%-40% |
| **临分/超赔** | 为大额风险提供保障，按层支付保费 | 层保费率 0.5%-5% |
| **合约再保** | 年度合约，固定条件，稳定可预测 | 综合成本 5%-15% |

### 3.2 定价中如何考虑再保
- **净费率** = 毛费率 - 再保分出成本
- 再保成本空间：通常预留 5%-15% 的费率空间
- 大额业务（超过自留额）需获得再保报价后再确定最终费率
- 创新险种通常需先获得再保支持才能定价

## 四、信用保证险定价（与财产险/责任险完全不同）

> checklist 步骤6仅覆盖了财产险/责任险/科技险的因子体系，此处补充信保类。

### 4.1 信保定价核心逻辑
信保险的定价核心是**违约概率（PD）× 违约损失率（LGD）× 违约风险暴露（EAD）**：
```
预期损失 = PD × LGD × EAD
保费 = 预期损失 + 费用 + 风险附加 + 利润
```

### 4.2 信保风险因子体系

| 因子类别 | 具体因子 | 权重 | 说明 |
|---------|---------|------|------|
| 信用风险 | 企业信用评级 | 35% | AAA 0.70 / AA 0.80 / A 0.90 / BBB 1.00 / BB 1.20 / B 1.50 |
| 交易结构 | 担保方式 | 25% | 足额抵押0.6 / 一般保证0.8 / 信用0.9 / 无担保1.5 |
| 行业风险 | 行业周期性 | 20% | 稳定行业0.7 / 周期行业1.0 / 高波动行业1.5 |
| 期限风险 | 保险期间 | 10% | ≤1年0.8 / 1-3年1.0 / >3年1.3 |
| 集中度 | 单一风险敞口 | 10% | ≤5%敞口0.8 / 5-10%1.0 / >10%1.5 |

### 4.3 信保定价注意事项
- **相关性风险**：经济下行时违约集中爆发，需加载系统性风险附加
- **逆选择风险**：投保人对自身风险的了解优于保险人
- **追偿收入**：理赔后可能通过追偿收回部分款项，可在定价中考虑追偿率
- **审批制**：信保业务通常需逐笔审批，费率因客户而异

## 五、现代精算方法（供参考）

### 5.1 GLM（广义线性模型）
- 用于分类费率厘定中因子筛选和系数估计
- 常用分布：泊松分布（频数）、伽马分布（ severity）
- 常用链接函数：对数链接

### 5.2 机器学习定价
- 随机森林、梯度提升树用于因子重要性排序
- 神经网络用于非线性风险模式识别
- **注意**：监管要求费率可解释性，纯黑箱模型难以通过审批

### 5.3 蒙特卡洛模拟
- 适用于低频高损风险（如巨灾保险、信用风险）
- 模拟大量损失场景，计算不同置信水平下的损失分布
- 输出：VaR、TVaR 等风险度量

## 六、准备金评估（精算报告必备章节）

### 6.1 准备金类型

| 类型 | 说明 | 评估方法 |
|------|------|---------|
| **未到期责任准备金** | 已收保费中尚未承担责任的部分 | 1/24法、1/365法、日比例法 |
| **未决赔款准备金** | 已报案尚未结案的赔款 | 逐案估计法（IBNR + RBNS） |
| **理赔费用准备金** | 处理赔案将发生的费用 | 按赔案比例或固定比例提取 |
| **巨灾准备金** | 应对潜在巨灾损失 | 按保费一定比例计提 |

### 6.2 评估要点
- 新产品首年准备金计提比例应保守
- 费用损失险的未到期责任准备金需考虑项目周期
- 信保险需额外计提信用风险准备金
- 准备金评估结果直接影响偿付能力充足率

## 七、费率审批备案的精算要求

### 7.1 精算报告必须包含
1. 定价方法说明及选择理由
2. 基础假设参数（赔付率、费用率、利润率）
3. 风险因子体系及权重依据
4. 基准费率的计算过程
5. 调整系数的设定依据
6. 费率合理性校验（与同业对比）
7. 准备金评估假设
8. 对公司偿付能力的影响评估

### 7.2 精算责任人声明要点
- 声明费率厘定方法合理、假设合理
- 声明费率充足性（覆盖预期赔款+费用+利润）
- 声明不存在侵害消费者权益的情形
