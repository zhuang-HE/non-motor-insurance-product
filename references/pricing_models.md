# 非车险定价模型与精算方法

## 一、定价基本原则

### 1.1 充足性原则
保费应足以覆盖：
- 预期赔款
- 理赔费用
- 承保费用
- 风险附加
- 利润附加

### 1.2 合理性原则
- 费率应与风险水平匹配
- 避免交叉补贴
- 考虑市场竞争

### 1.3 公平性原则
- 同风险同费率
- 风险差异应体现在费率中

## 二、定价方法

### 2.1 纯保费法
```
纯保费 = 预期赔款 / 风险单位数
毛保费 = 纯保费 / (1 - 费用率 - 利润率)
```

### 2.2 损失率法
```
指示费率 = 当前费率 × 经验损失率 / 目标损失率
经验损失率 = 已发生赔款 / 已赚保费
```

### 2.3 分类费率制定
```
基准费率 × 行业系数 × 地区系数 × 规模系数 × 管理系数 = 最终费率
```

## 三、风险因子体系

### 3.1 财产险风险因子
| 因子类别 | 具体因子 | 权重 |
|---------|---------|------|
| 标的特征 | 建筑结构、使用年限、占用性质 | 30% |
| 地理位置 | 自然灾害区域、消防距离 | 25% |
| 防护设施 | 消防系统、安防系统 | 20% |
| 管理水平 | 安全制度、历史损失 | 15% |
| 保险条件 | 免赔额、保险金额 | 10% |

### 3.2 责任险风险因子
| 因子类别 | 具体因子 | 权重 |
|---------|---------|------|
| 行业类型 | 风险等级分类 | 35% |
| 经营规模 | 营业额/员工数 | 20% |
| 区域差异 | 司法环境、赔偿标准 | 20% |
| 历史记录 | 索赔记录、安全管理 | 15% |
| 限额免赔 | 赔偿限额、免赔额 | 10% |

- 未决赔款准备金
