# 非车险定价模型与精算方法

## 一、定价基本原则

### 1.1 充足性原则
保费应足以覆盖：
- 预期赔款
- 理赔费用
- 承保费用
- 风险附加
- 利润附加

### 1.2 合理性原则
- 费率应与风险水平匹配
- 避免交叉补贴
- 考虑市场竞争

### 1.3 公平性原则
- 同风险同费率
- 风险差异应体现在费率中

## 二、定价方法

### 2.1 纯保费法
```
纯保费 = 预期赔款 / 风险单位数
毛保费 = 纯保费 / (1 - 费用率 - 利润率)
```

### 2.2 损失率法
```
指示费率 = 当前费率 × 经验损失率 / 目标损失率
经验损失率 = 已发生赔款 / 已赚保费
```

### 2.3 分类费率制定
```
基准费率 × 行业系数 × 地区系数 × 规模系数 × 管理系数 = 最终费率
```

## 三、风险因子体系

### 3.1 财产险风险因子
| 因子类别 | 具体因子 | 权重 |
|---------|---------|------|
| 标的特征 | 建筑结构、使用年限、占用性质 | 30% |
| 地理位置 | 自然灾害区域、消防距离 | 25% |
| 防护设施 | 消防系统、安防系统 | 20% |
| 管理水平 | 安全制度、历史损失 | 15% |
| 保险条件 | 免赔额、保险金额 | 10% |

### 3.2 责任险风险因子
| 因子类别 | 具体因子 | 权重 |
|---------|---------|------|
| 行业类型 | 风险等级分类 | 35% |
| 经营规模 | 营业额/员工数 | 20% |
| 区域差异 | 司法环境、赔偿标准 | 20% |
| 历史记录 | 索赔记录、安全管理 | 15% |
| 限额免赔 | 赔偿限额、免赔额 | 10% |

## 四、附加险定价模型

### 4.1 附加险定价原则
- 附加险费率通常低于主险（附加险的风险暴露是主险的子集或扩展）
- 附加险除外责任不得少于主险，因此风险更低
- 附加险定价可参照主险费率乘以附加系数，或完全独立定价

### 4.2 附加险定价方法

#### 方法一：主险费率比例法（推荐）
```
附加险费率 = 主险基准费率 × 附加系数
附加系数参考值：
  - 风险扩展型：0.2 ~ 0.5（扩展新风险，系数较高）
  - 费用扩展型：0.05 ~ 0.15（扩展费用项目，系数较低）
  - 限额扩展型：0.1 ~ 0.3（提高限额，系数中等）
  - 场景扩展型：0.1 ~ 0.3（特定场景扩展）
```

#### 方法二：独立定价法
```
适用于：附加险承保的风险与主险差异较大
步骤：
  1. 确定附加险的预期损失率（通常 30%-50%，低于主险）
  2. 选择定价方法（纯保费法/类比法）
  3. 设定调整因子（简化版，3-5个因子）
  4. 计算附加险基准费率
```

#### 方法三：行业协会参考法
```
适用于：行业协会已有标准费率的附加险
直接参照行业协会公布的附加险标准费率执行
```

### 4.3 附加险调整因子（简化版）
附加险通常只需 3-5 个调整因子（主险通常 5-8 个）：

| 附加险类型 | 常用调整因子 | 数量 |
|-----------|------------|------|
| 盗窃扩展 | 区域治安等级、防护设施、保险金额 | 3个 |
| 玻璃破碎 | 玻璃类型、建筑结构、面积 | 3个 |
| 清理残骸 | 保险金额、建筑类型、消防等级 | 3个 |
| 食物中毒责任 | 餐饮类型、座位数、卫生等级 | 3个 |
| 产品召回费用 | 产品类别、年销售额、召回概率 | 4个 |

### 4.4 附加险费率合理性校验
- [ ] 附加险费率 + 主险费率的综合费率偏离行业水平 ≤ ±30%
- [ ] 附加险费率不低于附加险预期赔付率对应的理论费率
- [ ] 附加险短期费率表与主险短期费率表一致
- [ ] 附加险数据底稿保存为 CSV 格式

### 4.5 附加险定价示例
**场景**：财产综合险附加盗窃扩展
- 主险基准费率：1.5‰
- 盗窃附加系数：0.3（风险扩展型，中间值）
- 附加险基准费率：1.5‰ × 0.3 = 0.45‰
- 调整因子：区域治安（0.8-1.2）、防护设施（0.7-1.1）、保险金额（0.9-1.0）
- 最终附加险费率：0.45‰ × 加权调整系数

## 六、精算报告数据底稿规范

> **核心要求**：精算报告中的每个费率数值、赔付率假设、风险因子权重，都必须能追溯到数据底稿。

### 6.1 数据底稿CSV统一格式

所有数据底稿统一使用 CSV 格式，存放于 `data_drafts/` 目录：

| 底稿编号 | 来源步骤 | 文件命名规范 | 内容说明 |
|---------|---------|------------|---------|
| Draft-01 | 步骤1 | `step1_industry_data_YYYYMMDD.csv` | 行业标的数据、灾害风险系数 |
| Draft-02 | 步骤2 | `step2_credit_rating_YYYYMMDD.csv` | 投保人信用评级、风险评估 |
| Draft-03 | 步骤3 | `step3_exemption_data_YYYYMMDD.csv` | 竞品除外责任、历史赔付参考 |
| Draft-04 | 步骤4 | `step4_market_data_YYYYMMDD.csv` | 市场规模、竞品分析、宏观数据 |
| Draft-05 | 步骤6 | `step6_pricing_data_YYYYMMDD.csv` | 行业费率、赔付率、精算假设 |
| Draft-06 | 步骤8 | `step8_rider_pricing_YYYYMMDD.csv` | 附加险定价参考数据 |

### 6.2 CSV字段规范

**有外部数据时**：
```csv
数据项名称,查询值,数据来源,查询时间,备注
行业平均赔付率,52.3%,精算数据库,2026-03-30,近3年平均
行业基准费率,1.2‰,行业协会年报,2026-03-30,财产综合险
```

**无外部数据时**（精算报告必须标注）：
```csv
数据项名称,数据值,数据来源,说明
行业平均赔付率,55%,精算师经验判断,保守估计
行业基准费率,1.5‰,类比法,参考XX公司同类产品
```

### 6.3 精算报告中的数据来源声明模板

精算报告正文中应包含以下数据来源声明章节：

```markdown
## 数据来源说明

### 外部数据
| 数据项 | 数据来源 | 底稿文件 |
|--------|---------|---------|
| 行业保费规模 | 行业协会年报2025 | data_drafts/step6_pricing_data_20260330.csv |
| 行业赔付率 | 精算数据库 | data_drafts/step6_pricing_data_20260330.csv |

### 经验判断数据
| 数据项 | 判断依据 | 判断依据说明 |
|--------|---------|------------|
| 风险因子权重 | 精算师经验 | 参照行业惯例 |
| 目标赔付率 | 公司历史数据 | 近3年平均 |

### 数据来源声明
本报告部分费率测算数据基于精算师经验判断和类比法确定，
主要参照[XXX公司同类产品]和[行业标准费率]。
由于缺乏自身历史赔付数据，首年费率采用保守定价策略，
预留调整空间。
```
