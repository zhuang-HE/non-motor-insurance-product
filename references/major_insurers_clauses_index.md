## 二、关键条款内容摘要

### 2.1 太平洋产品责任险条款

#### 除外责任（14项）
1. 投保人、被保险人或其代表的故意行为或重大过失
2. 战争、类似战争行为、敌对行为、军事行动、武装冲突、恐怖活动
3. 核裂变、核聚变、核武器、核材料、核辐射及放射性污染
4. 行政行为或司法行为
5. 罚款、罚金或惩罚性赔偿
6. 被保险人根据合同或协议应承担的责任（但即使没有该合同或协议，依法仍应承担的责任除外）
7. 被保险人或其雇员所有或管理的财产损失
8. 产品仍在制造或销售场所期间发生的损失
9. 产品退换、回收、召回
10. 产品本身损失
11. 精神损害赔偿
12. 间接损失
13. 大气、土地、水污染及其他各种污染
14. 保险合同中载明的免赔额或免赔率

#### 赔偿处理要点
- 事故发生之日起2年内提出索赔
- 诉讼时效3年
- 代位求偿权：保险人赔偿后取得向第三方的追偿权

### 2.5 紫金保险公众责任险条款

> **来源**：行业协会条款库 iachina.cn

#### 除外责任（9项原因除外 + 7项损失除外）

**原因除外：**
1. 火灾、地震、爆炸、洪水、烟熏
2. 大气、土地、水污染及其他污染
3. 有缺陷的卫生装置、中毒、不洁或有害食物或饮料
4. 被保险人做出的医疗措施或医疗建议
5. 震动、移动或减弱支撑引起的土地、财产、建筑物损坏
6. 战争、类似战争行为、敌对行为、武装冲突、恐怖活动、谋反、政变
7. 罢工、暴动、民众骚乱或恶意行为
8. 投保人、被保险人及其代表的故意行为或重大过失
9. 核裂变、核聚变、核武器、核材料、核辐射及放射性污染

**损失除外：**
1. 被保险人自有、保管或控制的财产
2. 被保险人一直使用和占用的物品、土地、房屋或建筑
3. 未载入保单的牲畜、车辆、船只、飞机、电梯、起重机等
4. 合同责任（无合同存在时仍应承担的责任除外）
5. 罚款、罚金或惩罚性赔款
6. 为被保险人服务的任何人所遭受的伤害
7. 保单中规定的免赔额

> **与其他公司对比**：紫金将"火灾、地震"列为除外（一般需特约扩展），而人保/平安/太平洋等公司在公众责任险中火灾通常可通过附加险扩展承保。

#### 赔偿限额结构（三层级，与行业一致）
| 层级 | 说明 |
|------|------|
| 第一层 | 每次事故赔偿限额 |
| 第二层 | 每人人身伤亡赔偿限额 |
| 第三层 | 累计赔偿限额 |

### 2.6 紫金保险金融机构责任险条款

> **来源**：行业协会条款库 iachina.cn

#### 保险责任（两类）
1. **营业大厅人身伤亡/财产损失**：金融机构正式在册职工和临时工因疏忽或过失导致储户或第三者在营业大厅内的人身伤亡
2. **暴力抢劫/劫持损失**：在营业大厅内发生暴力抢劫、劫持，导致储户或第三者人身伤亡以及储户当时所提取现金被抢夺的损失

## 七、生物医药领域保险条款

### 7.1 药物临床试验责任保险

#### 产品概览
| 公司名称 | 条款名称 | 适用范围 | 条款来源 |
|---------|---------|---------|---------|
| 平安产险 | 平安药物临床试验责任保险条款 | 药物临床试验申办方 | 行业协会 iachina.cn |
| 太平洋产险 | 药物临床试验责任保险（2023版） | 药物临床试验申办方 | 太平洋官网 cpic.com.cn |
| 太平洋产险 | 临床试验责任保险（B款） | 药物/医疗器械临床试验 | 太平洋官网 cpic.com.cn |
| 太平洋产险 | 生物医药人体临床试验责任保险 | 生物医药临床试验 | 太平洋官网 cpic.com.cn |
| 民安保险 | 药物临床试验责任险 | 药物临床试验 | 行业协会 iachina.cn |
| 中华联合 | 药物临床试验责任保险条款 | 药物临床试验 | 中华联合官网 cic.cn |
| 平安产险（上海） | 生物医药人体临床试验责任保险条款 | 生物医药临床试验 | 平安官网 iobs.pingan.com.cn |

#### 保险责任（通用设计）
1. **受试者人身伤亡赔偿**：在保险期间或追溯期内，因药物/器械不良反应出现严重不良事件，导致受试者遭受人身伤亡，依法应由被保险人承担的经济赔偿责任
2. **法律费用**：经保险人书面同意，因保险事故被提起仲裁或诉讼所产生的合理法律费用
3. **紧急医疗费用**（部分公司）：突发性、急需救治的受试者可预付部分医疗费用

#### 投保人/被保险人
- **主被保险人**：依法成立、具备药物/医疗器械临床试验申办资格的机构（申办方/Sponsor）
- **可扩展被保险人**（需经保险人同意）：
  - 研究者/主要研究者（PI）
  - 临床试验机构（医院）
  - 合同研究组织（CRO）
  - 伦理委员会
  - 参与试验的医生、护士

#### 责任免除（各公司共性除外项）
1. **试验合规性**：试验未获国家药监局/伦理委员会批准
2. **特定疾病**：试验涉及艾滋病、肝炎等相关病变
3. **药物制备**：试验药物制备不符合GMP质量管理规范
4. **行为不当**：被保险人或其雇员的故意行为/重大过失
5. **未遵守方案**：未遵守试验方案或GCP规范
6. **资质不足**：试验中心或研究者不具备资质
7. **境外试验**：试验在境外进行（除非特约扩展）
8. **非药物因素**：非试验药物/器械导致的不良事件
9. **自身疾病**：受试者原患疾病或先天性缺陷导致的伤亡
10. **精神疾病**：受试者患有精神疾病或急性中毒时导致的损失
11. **预期效果**：临床试验产品未达预期效果/预期副作用
12. **胎儿影响**：对孕妇、胎儿健康的影响（除非特约扩展）
13. **故意行为**：投保人或被保险人的故意行为或重大过失
14. **战争/核**：战争、恐怖活动、核辐射等
15. **精神损害**：精神损害赔偿
16. **间接损失**：间接损失
17. **罚款惩罚**：罚款、罚金或惩罚性赔偿

#### 赔偿限额结构
| 层级 | 说明 |
|------|------|
| 第一层 | 每次事故赔偿限额 |
| 第二层 | 每人人身伤亡赔偿限额 |
| 第三层 | 累计赔偿限额 |
| 分项 | 法律费用限额（通常为赔偿限额的一定比例，如10%-20%） |

#### 关键术语
| 术语 | 定义 |
|------|------|
| 严重不良事件(SAE) | 需住院、延长住院、伤残、危及生命或导致死亡等事件 |
| 知情同意书 | 受试者自愿参加试验的书面证明 |
| 临床试验 | 在人体进行药物/器械的系统性研究，以证实疗效与安全性 |
| GCP | 药物临床试验质量管理规范 |

### 7.2 临床试验责任保险（B款）——太平洋产险

> **来源**：太平洋产险官网 cpic.com.cn，2023年3月版

#### 保险责任
1. **第三者人身伤亡或财产损失**：因临床试验活动导致第三者人身伤亡或财产损失，依法应由被保险人承担的经济赔偿责任
2. **法律费用**：抗辩或解决索赔产生的必要法律费用（诉讼费、仲裁费、律师费等）
3. **紧急医疗费用**：突发性、急需救治的受试者可预付部分医疗费用

#### 特殊除外
- 试验药物/器械本身的质量问题（应通过产品责任险承保）
- 被保险人的雇员在雇佣期间遭受的伤害
- 未被批准的试验或超范围试验
- 试验前已明确告知受试者且受试者已签署知情同意书的风险

### 7.3 生物医药人体临床试验责任保险——平安产险（上海）

4. 临床试验产品对孕妇、胎儿健康的影响（但经保险人书面同意承保的不在此限）

### 8.2 网络安全保险保险责任（行业标准设计）

#### 第一部分：第三方责任保障
| 责任类型 | 保障内容 |
|---------|---------|
| **数据泄露责任** | 因安全事件导致的客户/个人信息泄露损失、未经授权披露 |
| **个人信息泄露** | 被保险人负责保管的非公开个人信息的盗取、丢失或未经授权披露 |
| **网络勒索** | 勒索软件攻击导致的赎金、恢复费用 |
| **法律费用** | 监管调查、客户诉讼的法律辩护费用 |
| **通知费用** | 根据法律法规向受影响个人/机构发出数据泄露通知的费用 |
| **危机公关** | 事件响应、声誉修复费用 |

#### 第二部分：第一方损失保障
| 责任类型 | 保障内容 |
|---------|---------|
| **营业中断** | 因网络攻击导致的业务中断利润损失 |
| **数据恢复** | 数据恢复、系统重建费用 |
| **事件响应** | 取证调查、事件管理费用 |
| **网络勒索** | 赎金支付、专业谈判费用 |

### 8.3 网络安全保险责任免除（各公司共性除外项）

#### 原因除外
1. **故意/欺诈行为**：投保人、被保险人或其雇员、IT外包商及其雇员的欺诈、违法行为、犯罪行为、重大过失或故意行为
2. **战争/恐怖活动**：战争、敌对行为、军事行动、武装冲突、恐怖活动
3. **核风险**：核裂变、核聚变、核武器、核辐射
4. **政府行为**：政府当局的没收、征用、销毁或破坏
5. **自然灾害**：地震、海啸、洪水、台风等不可抗力事件（部分公司可通过附加险扩展）

#### 损失除外
6. **固有缺陷**：信息系统自身的固有缺陷或已知漏洞
7. **业务中断间接损失**：被保险人自身业务中断、收入损失（除非条款另有约定）
8. **罚款/惩罚性赔偿**：政府、监管机构处以的罚款、罚金或惩罚性赔偿
9. **精神损害**：第三方提出的精神损害赔偿
10. **合同责任**：被保险人在合同中约定的责任（无法定责任的除外）
11. **升级费用**：改动或毁坏被保险人的信息技术设备、通讯设备或基础设施
12. **已预见索赔**：保险期间开始前已经知道或应当知道的索赔
13. **未维护安全**：未按照行业标准或安全规范维护网络安全

## 九、使用指引

1. **开发新产品时**：先查询同类型产品的已备案条款，参考其除外责任和赔偿处理设计
2. **设计除外责任时**：对照五大公司同类条款的除外责任，确保不遗漏关键除外项
3. **设计赔偿限额时**：参考行业标准限额结构，确保层次完整
4. **选择附加险时**：在行业协会附加险清单中筛选匹配的附加险组合
5. **条款合规审查时**：对照标准条款结构，确保13章节完整无遗漏
6. **开发生物医药保险时**：参考第七章各保司临床试验责任险条款，重点注意试验合规性免责、资质审查、索赔宽限期等特殊设计
7. **开发网络安全保险时**：参考第八章各保司网络安全险条款，注意第一方/第三方责任分层、已知漏洞除外、安全维护义务等特殊要求
8. **使用紫金保险条款时**：注意紫金公众责任险将火灾/地震列为绝对除外（区别于其他公司的特约扩展模式），附加险数量丰富（105个公众责任险附加险）
