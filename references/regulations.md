# 非车险监管政策与合规要求

## 一、产品审批备案制度

### 1.1 审批类产品
- 关系社会公众利益的保险险种
- 依法实行强制保险的险种
- 新开发的人寿保险险种

### 1.2 备案类产品（非车险主要类别）
- 财产保险公司开发的非车险产品
- 需向银保监会备案
- 备案材料：条款、费率、可行性报告

### 1.3 产品开发禁止性规定
1. 不得违反法律、行政法规
2. 不得损害社会公共利益
3. 不得侵害投保人、被保险人合法权益
4. 不得危及公司偿付能力

## 二、条款设计规范

### 2.1 条款必备要素
```
保险条款结构：
├── 总则（合同构成、投保人/保险人定义）
├── 保险标的
├── 保险责任
├── 责任免除
├── 保险价值/金额
├── 保险期间
├── 保险人义务
├── 投保人/被保险人义务
├── 赔偿处理
├── 争议处理
└── 释义
```

### 2.2 费率监管要求
- 费率不得畸高或畸低
- 需有精算依据
- 费率调整需重新备案
- 不得使用不公平、不合理条款

## 三、偿付能力管理
- 实际资本与最低资本要求
- 非车险业务资本占用
- 再保险安排对偿付能力的影响
