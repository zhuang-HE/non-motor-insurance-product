# 非车险案例库

> 保险产品条款、费率表、可研报告检索系统

## 概述

本仓库收录各类非车险产品条款文档，支持快速检索与查阅。

## 内容结构

```
cases/
├── product_index.md          # 产品总索引
├── 财产险/                   # 财产损失险
├── 责任险/                   # 责任保险
├── 意外险/                   # 意外伤害保险
├── 健康险/                   # 健康医疗保险
├── 信用保证险/              # 信用保险、保证保险
├── 工程险/                   # 建筑工程险、安装工程险
├── 特殊风险/                 # 无人机、机器人等特殊设备险
├── 货运险/                   # 货物运输险
├── 船舶险/                   # 船舶保险
├── 能源险/                   # 电力、热能等能源险
├── 农业险/                   # 农业保险
└── 网络安全保险/             # 网络安全保险
```

## 使用方式

### 本地检索

```bash
# 安装依赖
pip install python-docx pypdf pdfplumber

# 转换文档为md格式
python scripts/convert_clauses.py
```

### 在线查阅

直接在 GitHub 仓库中浏览 `cases/` 目录，或使用 `product_index.md` 查看产品总索引。
