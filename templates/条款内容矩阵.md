# 保险条款内容矩阵

> 按险种大类（监管分类）及条款板块组织的标准化条款内容模板库
> 用于快速生成符合监管要求的保险条款文档

---

## 一、险种大类分类（监管要求）

| 大类代码 | 大类名称 | 包含险种 | 说明 |
|---------|---------|---------|------|
| A | 财产保险 | 财产基本险、财产综合险、财产一切险、机器损坏险 | 含存货险、货架险等 |
| B | 责任保险 | 公众责任险、雇主责任险、产品责任险、安全生产责任险 | 含职业责任险、诉讼责任险 |
| C | 信用保证保险 | 信用险、保证保险 | 含投标保证、履约保证、关税保证 |
| D | 工程保险 | 建筑工程一切险、安装工程一切险 | 含工程机械设备险 |
| E | 货运险 | 国内水路/陆路货运险、海洋运输货物险、航空运输货物险 | 含集装箱险、冷藏货物险 |
| F | 特殊风险保险 | 无人机险、eVTOL险、网络安全险 | 科技保险核心领域 |
| G | 科技保险 | 首台套险、专利险、数据资产险、中试险 | 享受补贴政策的科技险 |
| H | 短期健康险 | 团体意外险、健康险 | 附加于财产险的意外医疗 |
| I | 其他险 | 食安险、环责险 | 政策引导型险种 |

---

## 二、条款板块结构（13章节）

```
第1章 总则          - 合同组成、术语定义
第2章 保险标的      - 可保标的、特约标的、不可保标的
第3章 保险责任      - 主险责任、附加险责任
第4章 责任免除      - 通用除外、产品特定除外
第5章 保险价值/保险金额/责任限额/免赔额（率）
第6章 保险期间
第7章 保险费
第8章 保险人义务
第9章 投保人、被保险人义务
第10章 赔偿处理
第11章 争议处理和法律适用
第12章 其他事项
第13章 释义
附录 短期费率表
```

## 三、板块内容矩阵

### 3.1 总则（第1章）矩阵

#### A类-财产险总则模板

| 条款项 | 标准表述 |
|-------|---------|
| 合同组成 | 本保险合同由保险条款、投保单、保险单或其他保险凭证以及批单组成。凡涉及本保险合同的约定，均应采用书面形式。 |
| 投保人定义 | 本保险合同的投保人应为对保险标的具有保险利益的自然人、法人或其他组织。 |
| 被保险人定义 | 本保险合同的被保险人为在保险事故发生时，对保险标的享有保险利益的人。 |
| 受益人 | 本保险合同不指定受益人，保险金给付给被保险人。 |

#### B类-责任险总则模板

| 条款项 | 标准表述 |
|-------|---------|
| 合同组成 | 本保险合同由保险条款、投保单、保险单或其他保险凭证以及批单组成。凡涉及本保险合同的约定，均应采用书面形式。 |
| 投保人定义 | 本保险合同的投保人应为依法设立并合法存续的企业、事业单位、国家机关、社会团体或其他组织。 |
| 被保险人定义 | 本保险合同的被保险人应为投保人或依法应承担本条款约定的赔偿责任的人。 |
| 赔偿责任 | 被保险人因保险合同列明的情形造成第三者人身伤亡或财产损失，依照中华人民共和国法律应由被保险人承担的经济赔偿责任，保险人按照本保险合同约定负责赔偿。 |

#### E类-货运险总则模板

| 条款项 | 标准表述 |
|-------|---------|
| 合同组成 | 本保险合同由保险条款、投保单、保险单或其他保险凭证以及批单组成。凡涉及本保险合同的约定，均应采用书面形式。 |
| 运输方式 | 本保险合同适用于[水路/陆路/航空/多式联运]运输方式。 |
| 运输区间 | 本保险合同保障的运输区间为从[起运地]至[目的地]的运输过程，包括中转、装卸、临时仓储等环节。 |
| 投保人定义 | 本保险合同的投保人应为对运输货物具有保险利益的发货人、收货人、承运人或其代理人。 |
| 被保险人定义 | 本保险合同的被保险人为保险单载明的被保险人。 |

#### F/G类-科技保险总则模板

| 条款项 | 标准表述 |
|-------|---------|
| 合同组成 | 本保险合同由保险条款、投保单、保险单或其他保险凭证以及批单组成。凡涉及本保险合同的约定，均应采用书面形式。 |
| 科技属性 | 本保险合同[适用/不适用]国家科技保险相关政策。 |
| 投保人定义 | 本保险合同的投保人应为经[相关政府部门]认定的高新技术企业、科技型中小企业或[其他认定资质]。 |
| 被保险人定义 | 本保险合同的被保险人应为[项目承担单位/企业/科研机构]。 |
| 认定依据 | 本保险合同所称[项目名称/专利/设备]以[立项文件/专利证书/购置发票]为准。 |

### 3.2 保险标的（第2章）矩阵

#### A类-财产险标的模板

| 条款项 | 标准表述 |
|-------|---------|
| 可保标的 | 本保险合同的保险标的是指[地点]内属于被保险人所有或与他人共有、由被保险人保管或与他人共有、由被保险人提供为自己或他人保管、或与他人协作照料使用的[房屋/机器设备/原材料/在产品/产成品/库存商品]等[财产/资产]。 |
| 特约标的 | 下列财产非经投保人与保险人特别约定，并在保险单上载明，不在保险标的范围以内：（一）金银、珠宝、钻石、玉器、首饰、古币、古玩、古书、字画、邮票、艺术品、稀有金属等珍贵财物；（二）堤堰、水闸、铁路、道路、涵洞、桥梁、码头；（三）矿井、矿坑的地下建筑物、设备和矿下物资；（四）[其他贵重物品]。 |
| 不可保标的 | 下列财产不在保险标的范围以内：（一）土地、矿藏、矿井、矿坑、森林、水产资源以及未经收割或收割后尚未入库的农作物；（二）货币、票证、有价证券、文件、账册、图表、技术资料、电脑资料、枪支弹药、爆炸物品、酒、草料、鲜活物品、烟叶、食品、药品、化妆品等（已约定为特约标的的除外）；（三）违章建筑、非法占用建筑或设施；（四）[其他不符合可保条件的财产]。 |

#### E类-货运险标的模板

| 条款项 | 标准表述 |
|-------|---------|
| 可保货物 | 本保险合同的保险标的是指在保险单中载明的，经[运输方式]运输的[货物名称/类别]，包括主货物及其附属包装、衬垫物。 |
| 鲜活货物 | 鲜活货物指[蔬菜/水果/鲜活动物/海产品/鲜花/苗木]等易腐、易变质或在运输过程中需要特殊保鲜条件的货物。鲜活货物须在投保时特别申报，经保险人同意后方可承保。 |
| 精密仪器 | 精密仪器指[医疗器械/科研仪器/电子设备/光学仪器]等价值高、易损坏的货物。精密仪器须符合[包装标准/装载要求]。 |
| 危险货物 | 危险货物指按照[国家标准GB6944《危险货物分类与品名编号》]界定的[爆炸品/压缩气体/易燃液体/易燃固体/氧化剂/毒害品/放射性/腐蚀品]等。危险货物须持有有效的运输许可证，并按规定的包装和标记进行运输。 |
| 贵重物品 | 贵重物品指单件价值超过[人民币X万元]的[珠宝/贵金属/艺术品/古董/精密仪器]。贵重物品须在投保时申报价值并经保险人核保同意。 |
| 不保货物 | 下列货物不在保险标的范围以内：（一）金银、珠宝、钻石、玉器、首饰、货币、票证、有价证券、邮票、古币、古玩、古书、字画、艺术品；（二）武器弹药、爆炸物品；（三）鲜活货物（已约定为特约标的的除外）；（四）违禁品及非法运输物品。 |

#### F/G类-科技保险标的模板

| 条款项 | 标准表述 |
|-------|---------|
| 费用损失险标的 | 本保险合同的保险标的是指被保险人在[项目名称]研发/转化/中试过程中实际投入的[研发费用/材料费/设备费/人工费/试验费/评审费/其他直接费用]，以保险单载明的金额为准。 |
| 专利执行险标的 | 本保险合同的保险标的是指被保险人依法拥有的，经[国家知识产权局]授权且在有效保护期内的[发明专利/实用新型专利/外观设计专利]，以保险单载明的专利清单为准。 |
| 设备损坏险标的 | 本保险合同的保险标的是指被保险人所有或保管的，用于[研发/生产/测试]用途的[设备名称/型号]，以保险单载明的设备清单为准。 |
| 数据资产险标的 | 本保险合同的保险标的是指被保险人合法拥有或控制的，以电子数据形式存储的[数据类型]，包括[结构化数据/非结构化数据/数据库/软件代码/数字内容]等。 |

### 3.3 保险责任（第3章）矩阵

#### A类-财产险一切险责任模板

| 条款项 | 标准表述 |
|-------|---------|
| 主险责任 | 在保险期间内，由于自然灾害或意外事故造成保险标的的直接损失，保险人按照本保险合同的约定负责赔偿。 |
| 一切险范围 | 本保险合同所称\"自然灾害或意外事故\"包括：火灾、爆炸、雷击、暴风、暴雨、洪水、台风、龙卷风、泥石流、崖崩、滑坡、地面突然塌陷、飞行物体及其他空中运行物体坠落，以及盗窃、抢劫、故意毁坏等人为行为。 |
| 施救费用 | 保险事故发生后，被保险人为防止或减少保险标的的损失所支付的必要的、合理的费用（以下简称\"施救费用\"），保险人按照本保险合同的约定负责赔偿。 |

#### E类-货运险责任模板

| 条款项 | 标准表述 |
|-------|---------|
| 一切险责任 | 保险货物在运输过程中因遭受下列自然灾害或意外事故造成的直接损失，保险人按照本保险合同的约定负责赔偿：（一）火灾、爆炸；（二）运输工具遭受搁浅、触礁、沉没、碰撞；（三）碰撞、倾覆、坠落；（四）避难港卸载或在中途港卸载；（五）共同海损牺牲、救助费用；（六）自然灾害造成的货物整批失踪。 |
| 综合险责任 | 综合险是在基本险的基础上，增加下列自然灾害或意外事故造成的损失：（一）暴雨、洪水、台风、暴风、龙卷风；（二）地震、海啸；（三）雷电、雹灾、雪灾、冰凌；（四）山崩、滑坡、泥石流。 |
| 基本险责任 | 基本险承保下列原因造成货物的损失：（一）火灾、爆炸；（二）运输工具发生碰撞、出轨、倾覆、坠落；（三）隧道坍塌、桥梁断裂、道路损坏。 |
| 施救费用 | 保险事故发生后，被保险人为防止或减少货物损失所支付的必要的、合理的施救费用，保险人按照本保险合同的约定负责赔偿。 |
| 额外费用 | 保险事故发生后，被保险人为查明和确定保险事故的性质、原因和损失程度所支付的必要的、合理的检验、鉴定、评估费用（以下简称\"检验费用\"），保险人按照本保险合同的约定负责赔偿。 |

#### F/G类-科技保险责任模板

| 条款项 | 标准表述 |
|-------|---------|
| 费用损失险责任 | 在保险期间内，被保险人按照[项目合同/研发计划]开展的[项目名称]，因下列原因导致项目失败或终止，造成被保险人实际投入的费用损失，保险人按照本保险合同的约定负责赔偿：（一）项目技术不成熟，无法达到预期技术指标；（二）项目关键人员离职或丧失行为能力；（三）主要设备故障或损毁；（四）不可抗力导致项目无法继续；（五）[其他特定原因]。 |
| 专利执行险责任 | 在保险期间内，被保险人发现在中华人民共和国境内有第三方侵犯其[专利权/专利申请权]，并就上述侵权行为采取以下维权行动的，保险人按照本保险合同的约定负责赔偿被保险人因维权行动产生的合理费用：（一）调查取证费；（二）律师费；（三）仲裁或诉讼费；（四）行政投诉处理费。 |
| 设备损坏险责任 | 在保险期间内，保险标的发生下列原因造成的直接物质损失或损坏，保险人按照本保险合同的约定负责赔偿：（一）设计错误或原材料缺陷；（二）制造缺陷或安装工艺不当；（三）操作失误或违规操作；（四）电气短路、过载或电网故障；（五）离心力导致的断裂；（六）[其他意外事故]。 |
| 认定程序 | 保险事故发生后，须由[第三方专业机构/政府科技部门/行业主管部门]出具[项目失败/专利侵权/设备损坏]认定报告，作为保险理赔的依据。 |

### 3.4 责任免除（第4章）矩阵

#### 通用除外条款（所有险种必须包含）

| 序号 | 除外项 | 标准条款表述 |
|-----|-------|------------|
| G1 | 战争/军事/恐怖 | 因战争、军事行动、武装冲突、恐怖活动、暴乱、罢工、骚乱、哄抢、扣押、征用、没收、政府行为等原因造成的损失、费用和责任，保险人不负责赔偿。 |
| G2 | 核辐射/污染 | 因核辐射、核爆炸、核污染及其他放射性污染造成的损失、费用和责任，保险人不负责赔偿。 |
| G3 | 行政/司法行为 | 因行政行为或司法行为造成的损失、费用和责任，保险人不负责赔偿。 |
| G4 | 故意/重大过失 | 投保人或被保险人的故意行为、重大过失或犯罪行为造成的损失、费用和责任，保险人不负责赔偿。 |
| G5 | 自然磨损/内在缺陷 | 保险标的因自然磨损、自然损耗、自然蒸发、氧化、锈蚀、锈坏、变质、破损、褪色、渗漏、鼠咬、虫蛀、机械或电气故障、内在缺陷造成的损失和费用，保险人不负责赔偿。 |
| G6 | 间接损失 | 因保险事故造成的间接损失、预期利润损失、违约金、罚款、罚金及惩罚性赔偿，保险人不负责赔偿。 |

#### A类-财产险特定除外

| 序号 | 除外项 | 标准条款表述 |
|-----|-------|------------|
| P1 | 正常消耗 | 保险标的因正常磨损、锈蚀、氧化、变质等渐变性原因造成的损失，保险人不负责赔偿。 |
| P2 | 保管不善 | 被保险人未按安全规程操作或未采取合理防护措施造成的损失，保险人不负责赔偿。 |
| P3 | 停产停业损失 | 因保险事故造成的停产、停业期间的间接损失（包括利润损失、费用增加等），保险人不负责赔偿。 |

#### E类-货运险特定除外

| 序号 | 除外项 | 标准条款表述 |
|-----|-------|------------|
| T1 | 包装不当 | 货物包装不符合运输要求，或包装完好但内件短少，保险人不负责赔偿。 |
| T2 | 自然损耗 | 货物因自然损耗、自然减量、破碎、渗漏、沾污、锈蚀、潮霉、虫蛀等造成的损失，保险人不负责赔偿。 |
| T3 | 延迟损失 | 任何原因造成的运输延迟、货物腐烂、变质、变质、退关退货损失，以及因此产生的费用和责任，保险人不负责赔偿。 |
| T4 | 承运人责任 | 因承运人的故意行为或重大过失造成的损失，保险人不负责赔偿；但被保险人不知情或无法控制的除外。 |
| T5 | 超额装载 | 货物装载超过运输工具的额定载重或空间限制造成的损失，保险人不负责赔偿。 |
| T6 | 战争险除外 | 本保险不负责赔偿战争、军事行动、扣押、没收、罢工、核风险等（可通过附加战争险扩展）。 |
| T7 | 装卸损失 | 货物在装卸过程中因操作不当造成的损失（可通过附加险扩展）。 |

#### F/G类-科技保险特定除外

| 序号 | 除外项 | 标准条款表述 |
|-----|-------|------------|
| S1 | 市场风险 | 因市场价格波动、商业判断失误、市场需求变化等市场因素造成的损失，保险人不负责赔偿。 |
| S2 | 研发失败正常风险 | 因研发项目技术路线选择失误、技术方案不优等属于正常技术风险的损失，保险人不负责赔偿。 |
| S3 | 知识产权侵权自侵 | 因被保险人侵犯他人知识产权（包括专利权、商标权、著作权等）造成的损失，保险人不负责赔偿。 |
| S4 | 违约责任 | 被保险人因违反合同约定应承担的违约金、罚款、赔偿金等合同责任，保险人不负责赔偿。 |
| S5 | 政府行为/政策变化 | 因政府产业政策调整、财政补贴政策变化、行业准入政策变化等政府行为造成的损失，保险人不负责赔偿。 |
| S6 | 已知缺陷/漏洞 | 保险责任开始前被保险人已知的缺陷、漏洞、侵权行为或纠纷，保险人不负责赔偿。 |
| S7 | 行政处罚 | 因被保险人违反法律法规被处以的行政罚款、刑事罚金及惩罚性赔偿，保险人不负责赔偿。 |

### 3.5 金额/限额/免赔（第5章）矩阵

#### A类-财产险金额模板

| 条款项 | 标准表述 |
|-------|---------|
| 保险价值 | 保险价值为保险标的在保险责任开始时的实际价值，即[重置价值/账面余额/市场价值]。重置价值是指在出险地重新购置或建造同一类型、同一规格、同一新旧程度财产所需的全部费用。实际价值是指保险合同生效时保险标的的市场价值或账面余额。 |
| 保险金额确定 | 保险金额由保险合同双方参照保险价值协商确定，并在保险合同中载明。保险金额不得超过保险价值，超过保险价值的，超过部分无效，保险人应当退还相应的保险费。 |
| 不足额投保 | 保险金额低于保险价值时，保险人按保险金额与保险价值的比例计算赔偿。 |
| 赔偿限额 | 每次事故赔偿限额和累计赔偿限额由投保人与保险人协商确定，并在保险合同中载明。 |
| 免赔额/率 | 每次事故免赔额或免赔率由投保人与保险人在订立保险合同时协商确定，并在保险合同中载明。 |

#### E类-货运险金额模板

| 条款项 | 标准表述 |
|-------|---------|
| 保险价值确定 | 货物保险价值为起运地发票价格加运费、保险费及其他费用。贵重物品须单独申报价值。 |
| 发票价值 | 以发票价格为基础确定的保险金额，最高不得超过发票价格。 |
| 推定全损 | 保险货物发生保险事故后，实际全损已经不可避免，或受损货物的数量、品种、价值恢复所需费用之和超过保险价值时，可视为推定全损。 |
| 免赔额/率 | 每次事故免赔额[XX元]或免赔率[XX%]，以高者为准。鲜活货物、易碎品另行约定。 |

#### F/G类-科技保险金额模板

| 条款项 | 标准表述 |
|-------|---------|
| 保险金额确定 | 保险金额为被保险人在保险期间内实际投入的[研发费用/专利维权费用/设备购置费]总额，以保险单载明的金额为准。 |
| 费用范围 | 本保险合同所称\"费用\"包括：[材料费/设备费/人工费/试验费/外协费/差旅费/评审费/咨询费/法律费用/调查费用]，以合法有效的发票或支出凭证为准。 |
| 费用扣除 | 政府补贴、专项资金、合作方投入及其他已获得的补偿应从保险金额中扣除。 |
| 赔偿比例 | 保险赔偿以被保险人实际未获补偿的费用为限，且不超过保险单载明的保险金额。 |
| 免赔额/率 | 每次事故免赔额[XX元]或免赔率[XX%]。 |

### 3.6 保险期间（第6章）矩阵

#### A类-财产险期间模板

| 条款项 | 标准表述 |
|-------|---------|
| 保险期间 | 除另有约定外，保险期间为一年，以本保险合同载明的起讫时间为准。 |
| 追溯期 | 本保险合同不设追溯期，保险责任自保险期间起始之日零时开始。 |
| 提前终止 | 保险标的全损赔偿后，保险责任终止，保险人退还剩余期间的保费。 |

#### E类-货运险期间模板

| 条款项 | 标准表述 |
|-------|---------|
| 运输期间 | 本保险合同采用\"仓至仓\"条款，自货物离开起运地仓库或储存处所开始运输时生效，直至货物到达目的地收货人的仓库或储存处所时终止。 |
| 临时仓储 | 运输过程中在转运港或目的港的临时仓储期间，最长不超过[30]天。 |
| 分批运输 | 分批装运的货物，须分别确定保险金额，保险期间分别计算。 |
| 提前终止 | 货物在中途港被卸货、转让或退运，保险责任终止。 |

#### F/G类-科技保险期间模板

| 条款项 | 标准表述 |
|-------|---------|
| 项目期间 | 保险期间与[项目实施周期/专利有效期间/设备使用期间]一致，自[起始日期]起至[终止日期]止，最长不超过[一年/项目完成]。 |
| 等待期 | 本保险合同设[30/60/90]天等待期，自保险合同生效之日起计算，等待期内发生的保险事故，保险人不负责赔偿。 |
| 延长期间 | 经保险人书面同意，保险期间可延长[XXX]天。 |
| 认定期间 | 项目失败或专利侵权的认定，须在保险期间内或保险期间届满后[30/60]天内完成。 |

### 3.7 保险人义务（第7章）矩阵

| 条款项 | 标准表述 |
|-------|---------|
| 条款说明 | 订立保险合同时，采用保险人提供的格式条款的，保险人向投保人提供的投保单应当附格式条款，保险人应当向投保人说明保险合同的内容。对保险合同中免除保险人责任的条款，保险人在订立合同时应当在投保单、保险单或者其他保险凭证上作出足以引起投保人注意的提示，并对该条款的内容以书面或者口头形式向投保人作出明确说明；未作提示或者明确说明的，该条款不产生效力。 |
| 签发保单 | 本保险合同成立后，保险人应当及时向投保人签发保险单或其他保险凭证。 |
| 资料补充通知 | 保险人按照第二十九条的约定，认为被保险人提供的有关索赔的证明和资料不完整的，应当及时一次性通知投保人、被保险人补充提供。 |
| 核定责任 | 保险人收到被保险人的赔偿保险金的请求后，应当及时作出是否属于保险责任的核定；情形复杂的，应当在三十日内作出核定，但保险合同另有约定的除外。保险人应当将核定结果通知被保险人；对属于保险责任的，在与被保险人达成赔偿保险金的协议后十日内，履行赔偿保险金义务。 |
| 先予支付 | 保险人自收到赔偿保险金的请求和有关证明、资料之日起六十日内，对其赔偿保险金的数额不能确定的，应当根据已有证明和资料可以确定的数额先予支付。 |
| 保密义务 | 保险人对在办理保险业务中知道的投保人、被保险人的商业秘密及个人隐私，负有保密的义务。 |

### 3.8 投保人/被保险人义务（第8章）矩阵

| 条款项 | 标准表述 |
|-------|---------|
| 如实告知 | 订立保险合同，保险人就保险标的或者被保险人的有关情况提出询问的，投保人应当如实告知。投保人故意或者因重大过失未履行前款规定的如实告知义务，足以影响保险人决定是否同意承保或者提高保险费率的，保险人有权解除合同。 |
| 交付保费 | 除本保险合同另有约定外，投保人应当在保险责任起始日前一次性交清全部保险费。投保人未按约定交纳保险费的，本保险合同不生效。 |
| 安全维护 | 投保人、被保险人应当遵守国家有关消防、安全、生产操作、劳动保护等方面的规定，建立、健全安全生产制度，切实保障保险标的的安全。投保人、被保险人未按照约定履行其对保险标的的安全应尽责任的，保险人有权要求增加保险费或者解除合同。 |
| 危险增加通知 | 在保险期间内，保险标的转让、保险标的的使用性质或范围改变、保险标的的占用性质改变等原因导致保险标的危险程度显著增加的，被保险人应当及时书面通知保险人，保险人可以按照合同约定增加保险费或者解除合同。 |
| 出险通知 | 知道保险事故发生后，被保险人应当：（一）尽力采取必要、合理的措施，防止或减少损失；（二）及时通知保险人，并在[24/48/72]小时内向保险人书面说明事故发生的原因、经过和损失情况；（三）保护事故现场，允许并且协助保险人进行事故调查。 |
| 单证提供 | 被保险人请求赔偿时，应向保险人提供下列证明和资料：（一）保险单正本及批单正本；（二）索赔申请；（三）[事故证明/鉴定报告/损失清单/发票]等。 |

### 3.9 赔偿处理（第9章）矩阵

| 条款项 | 标准表述 |
|-------|---------|
| 赔偿方式 | 保险标的发生保险责任范围内的损失，保险人有权选择下列方式赔偿：（一）货币赔偿；（二）实物赔偿；（三）实际修复。具体赔偿方式由保险人根据损失情况决定。 |
| 全损赔偿 | 保险标的构成推定全损的，保险人按保险金额全额赔偿；保险金额高于保险价值时，以不超过保险价值计算赔偿。 |
| 部分损失 | 部分损失的赔偿金额以实际发生的损失、费用为准，但保险金额低于保险价值时，按保险金额与保险价值的比例计算赔偿。 |
| 施救费用 | 被保险人为防止或减少保险标的的损失所支付的必要的、合理的施救费用，在保险标的损失赔偿金额之外另行计算，但赔偿金额以不超过保险金额为限。 |
| 残值处理 | 保险标的遭受损失后，如果有残余价值，应由双方协商处理。如折归被保险人，由双方协商确定其价值，并在保险赔款中扣除。 |
| 重复保险 | 存在重复保险时，保险人按照本保险合同的相应保险金额与所有有关保险合同的相应保险金额总和的比例承担赔偿责任。 |
| 代位求偿 | 因第三方的行为造成保险事故，保险人自向被保险人赔偿保险金之日起，在赔偿金额范围内依法代位行使被保险人对有关责任方请求赔偿的权利。 |
| 政府补偿扣除 | 被保险人已从政府财政补贴、专项扶持资金或第三方赔偿中获得的补偿，应在保险赔偿金额中扣除。 |
| 诉讼时效 | 被保险人向保险人请求赔偿保险金的诉讼时效期间为二年，自其知道或者应当知道保险事故发生之日起计算。 |

### 3.10 争议处理（第10章）矩阵

|-------|---------|
| 争议解决 | 因履行本保险合同发生的争议，由当事人协商解决；协商不成的，提交[仲裁机构名称]仲裁或向[保险人所在地]人民法院提起诉讼。 |
| 法律适用 | 与本保险合同有关的以及履行本保险合同产生的一切争议，适用于中华人民共和国法律（不含港、澳、台地区法律）。 |

### 3.11 其他事项（第11章）矩阵

| 条款项 | 标准表述 |
|-------|---------|
| 合同终止 | 保险标的发生全部损失，属于保险责任的，保险人在履行赔偿义务后，本保险合同终止；不属于保险责任的，本保险合同终止，保险人按短期费率表计收自保险责任开始之日起至损失发生之日止期间的保险费，并退还剩余部分保险费。 |
| 部分损失退保 | 保险标的发生部分损失，自保险人赔偿之日起三十日内，投保人可以解除合同，保险人也可以解除合同。合同解除的，对于保险标的未受损失部分的保险费，保险人按短期费率表计收，退还剩余部分保险费。 |
| 退保计算 | 保险责任开始后，投保人要求解除合同的，应当向保险人支付保险合同生效日至解除日期间的保险费，剩余部分退还投保人。 |
| 合同转让 | 保险标的转让的，被保险人或者受让人应当及时通知保险人，经保险人同意后，本保险合同继续有效。 |

### 3.12 释义（第12章）矩阵

| 术语 | 定义 |
|-----|------|
| 火灾 | 火灾指在时间或空间上失去控制的燃烧所造成的灾害。须具有下列三个条件：（1）有燃烧现象；（2）偶然、意外发生的燃烧；（3）失去控制并有蔓延扩大的趋势。 |
| 爆炸 | 爆炸指物体在瞬间分解或燃烧时释放出大量气体和能量，并急剧膨胀的现象或过程。 |
| 雷击 | 雷击指雷电造成的直接击中或感应产生的电涌损害。 |
| 暴风 | 暴风指风速在17.2米/秒（相当于八级风力）以上的风。 |
| 暴雨 | 暴雨指每小时降雨量达16毫米以上，或连续24小时降雨量达50毫米以上的降雨。 |
| 台风 | 台风指中心附近最大风力达12级以上的热带气旋。 |
| 地震 | 地震指地壳发生的震动，包括构造地震、火山地震、陷落地震。 |
| 地面突然塌陷 | 地面突然塌陷指地壳因为自然变异或地下空洞突然塌陷造成的地面标高突然降低的现象。 |
| 飞行物体坠落 | 飞行物体坠落指从飞行器、陨石坠落造成的损失，以及固定在建筑物上的建造物、搁置物、悬挂物发生倒塌、坠落造成的损失。 |
| 第三者 | 第三者指被保险人及其雇员、代理人以外的人。 |
| 保险事故 | 保险事故指本保险合同约定的保险责任范围内的事故。 |
| 重置价值 | 重置价值指在出险地重新购置或建造同一类型、同一规格、同一新旧程度财产所需的全部费用。 |
| 实际价值 | 实际价值指保险合同生效时保险标的的市场价值或账面余额扣除折旧后的金额。 |
| 全损 | 全损指保险标的完全损毁、灭失，或受损程度已达到无法修复的程度，或修复费用超过保险价值。 |
| 推定全损 | 推定全损指保险标的的实际全损已经不可避免，或修复费用超过保险价值。 |

## 四、附加险条款结构

附加险作为主险的扩展，须独立成文，并严格遵循以下结构：

### 附加险13章节结构

```
第1条 总则
第2条 保险标的（适用时）
第3条 保险责任
第4条 责任免除
第5条 保险金额/责任限额与免赔额（率）
第6条 保险期间
第7条 赔偿处理
第8条 投保人/被保险人义务
第9条 合同解除
第10条 释义
第11条 其他事项
```

> 注意：附加险的"总则"须明确说明与主险的关系及适用范围。
