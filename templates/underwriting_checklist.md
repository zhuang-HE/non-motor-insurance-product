# 非车险核保检查清单

## 一、投保人/被保险人资格审查

### 1.1 基本资格审查
- [ ] 投保人具有完全民事行为能力
- [ ] 投保人对保险标的具有保险利益
- [ ] 被保险人身份明确
- [ ] 投保人与被保险人关系清晰

### 1.2 企业投保人
- [ ] 营业执照有效
- [ ] 统一社会信用代码正确
- [ ] 经营范围与保险标的相符
- [ ] 注册资本符合要求
- [ ] 成立时间超过1年（或符合特别规定）

### 1.3 信用状况审查
- [ ] 无不良信用记录
- [ ] 无保险欺诈历史
- [ ] 无重大未决诉讼
- [ ] 无欠缴保费记录

## 二、保险标的审查

### 2.1 标的合法性
- [ ] 标的来源合法
- [ ] 标的不属于违禁品
- [ ] 标的不属于非法建筑
- [ ] 标的权属清晰

### 2.2 标的可保性
- [ ] 标的可以用货币估价
- [ ] 标的风险可以转移
- [ ] 损失具有偶然性
- [ ] 标的不在不可保清单内

### 2.3 标的风险评估
- [ ] 地理位置风险适中
- [ ] 建筑结构符合要求
- [ ] 消防设施完备
- [ ] 安全管理规范

## 三、保险金额/赔偿限额审查

### 3.1 保险金额合理性
- [ ] 保险金额不超过保险价值
- [ ] 保险金额不低于最低要求
- [ ] 分项保额设置合理
- [ ] 保额与风险匹配

### 3.2 赔偿限额适当性
- [ ] 每次事故限额合理
- [ ] 累计赔偿限额充足
- [ ] 每人赔偿限额适当
- [ ] 限额与保费匹配

## 四、承保条件审查

### 4.1 免赔额（率）
- [ ] 绝对免赔额设置合理
- [ ] 绝对免赔率设置合理
- [ ] 免赔条件明确
- [ ] 免赔与费率匹配

### 4.2 保险期间
- [ ] 保险期间明确
- [ ] 起保日期合理
- [ ] 短期保险符合规定
- [ ] 续保条件清晰

### 4.3 特别约定
- [ ] 特别约定必要
- [ ] 特别约定合法
- [ ] 特别约定明确
- [ ] 客户已充分理解

## 五、风险查勘要求

### 5.1 必查项目（满足以下任一条件）
- [ ] 保额超过500万元
- [ ] 属于高风险行业
- [ ] 历史赔付率高
- [ ] 特殊标的
- [ ] 核保人认为必要

### 5.2 查勘内容
- [ ] 标的实际状况
- [ ] 周边环境风险
- [ ] 消防设施检查
- [ ] 安全管理评估
- [ ] 历史损失了解
- [ ] 拍照留档

## 六、再保险安排审查

### 6.1 自留额检查
- [ ] 保额在自留额范围内，或
- [ ] 已安排再保险

### 6.2 再保险安排
- [ ] 再保险方案合理
- [ ] 再保险对手方资质良好
- [ ] 再保险合同有效
- [ ] 手续费安排合理

## 七、承保权限审查

### 7.1 权限检查
- [ ] 在核保人权限范围内，或
- [ ] 已按权限上报审批

### 7.2 超权限业务
- [ ] 已提交上级审批
- [ ] 已附风险评估报告
- [ ] 已附查勘报告
- [ ] 审批意见明确

## 八、单证完整性审查

### 8.1 投保资料
- [ ] 投保单填写完整
- [ ] 投保人签章齐全
- [ ] 标的清单完整
- [ ] 相关证明文件齐全

### 8.2 风险评估资料
- [ ] 风险查勘报告（如需要）
- [ ] 标的影像资料
- [ ] 历史损失记录
- [ ] 其他支持材料

## 九、系统录入检查

### 9.1 基本信息
- [ ] 投保人信息准确
- [ ] 被保险人信息准确
- [ ] 标的地址准确
- [ ] 联系方式有效

### 9.2 保险责任
- [ ] 险种选择正确
- [ ] 责任范围准确
- [ ] 附加险录入正确
- [ ] 特别约定录入完整

### 9.3 保费信息
- [ ] 保险金额准确
- [ ] 费率适用正确
- [ ] 折扣计算准确
- [ ] 保费计算正确
