# 团体住院医疗费用保险（2023版）费率表

太平科技保险股份有限公司
团体住院医疗费用保险（2023版）费率表

- 一、基准赔付标准和年基准保险费
- （一）基准赔付标准
- 1.基准保险金额：
- （1）一般医疗费用保险金额：100万元。
- （2）重大疾病医疗费用保险金额：在一般医疗费用保险金额基础上叠加100 万元。
- （3）恶性肿瘤特定药品费用、质子重离子医疗费用：与一般医疗费用保险金及重大疾病医疗费用保险金共享保险金额。
- （4）重大疾病住院津贴保险金额：18000元。重大疾病住院津贴保险金额等于每日重大疾病住院津贴金额乘以180，每日重大疾病住院津贴的基准金额为100元。
- （5）重大疾病异地转诊交通费用保险金额：10000元。
- 2.基准等待期：30天。
- 3.基准免赔额/免赔天数：
- （1）一般医疗费用保险责任免赔额：1万元。
- （2）重大疾病医疗费用保险责任免赔额：0元。
- （3）恶性肿瘤特定药品费用保险责任免赔额：0元。
- （4）质子重离子医疗费用保险责任免赔额：0元。
- （5）重大疾病住院津贴保险责任免赔天数：0天。
- （6）重大疾病异地转诊交通费用保险责任免赔额：0元。
- 4.基准给付比例：
- （1）一般医疗费用保险金、重大疾病医疗费用保险金：
- ①若被保险人以参加基本医疗保险（或公费医疗，下同）身份投保，并以基本医疗保险身份就诊并结算：100%。
- ②若被保险人以参加基本医疗保险身份投保，但未以基本医疗保险身份就诊并结算：60%。
- ③若被保险人未以参加基本医疗保险身份投保：100%。
- （2）恶性肿瘤特定药品费用保险金：
- ①若被保险人以参加基本医疗保险身份投保：
- 若药品为基本医疗保险药品目录内且已经过基本医疗保险报销：100%。
- 若药品为基本医疗保险药品目录内但未经过基本医疗保险报销：60%。
- 若药品为基本医疗保险药品目录外：100%。
- ②若被保险人未以参加基本医疗保险身份投保，则基本医疗保险药品目录内药品和基本医疗保险药品目录外药品的给付比例均为100%。
- （3）质子重离子医疗费用保险金：100%。
- （4）重大疾病异地转诊交通费用保险金：80%。
- 5.基准保险期间：1年。
- （二）年基准保险费（单位：人民币元）
- 1.一般医疗费用和重大疾病医疗费用年基准保险费
- 注：1.0周岁指出生满30日且已健康出院；

- 注：1.0周岁指出生满30日且已健康出院；
- 2.66周岁及以上年龄段年基准保险费仅适用于续保。
- 3.质子重离子医疗费用年基准保险费
- 注：1.0周岁指出生满30日且已健康出院；
- 2.66周岁及以上年龄段年基准保险费仅适用于续保。
- 4.重大疾病住院津贴年基准保险费
- 注：1.0周岁指出生满30日且已健康出院；

- 5.重大疾病异地转诊交通费用年基准保险费

- 二、产品参数调整系数
- 产品参数调整系数为以下各项参数调整系数的乘积。
- （一）保险金额调整系数
- 根据本保险的费率结构及保险费计算逻辑，设置保险金额调整系数，保险金额越高，保险公司面临的风险越大。
- 保险金额调整系数1（一般医疗费用保险责任及重大疾病医疗费用保险责任、恶性肿瘤特定药品费用保险责任、质子重离子医疗费用保险责任适用）
- 保险金额调整系数2（重大疾病异地转诊交通费用保险责任适用）
- （二）免赔额调整系数（一般医疗费用保险责任及重大疾病医疗费用保险责任适用）
- 根据本保险的费率结构及保险费计算逻辑，设置免赔额调整系数，免赔额越高，保险公司面临的风险越低。免赔额调整系数为以下各项免赔额调整系数之积。
- 1.一般医疗费用保险责任免赔额调整系数
- 注：如果免赔额介于上述相邻免赔额之间，调整系数采用线性插值的方式计算。免赔额为30000元以上的按20000-30000元免赔额的斜率外推线性插值计算，调整系数最低不低于0.1。
- 2.重大疾病医疗费用保险责任免赔额调整系数
- （三）免赔天数调整系数（重大疾病住院津贴保险责任适用）
- （四）等待期调整系数
- 根据本保险的费率结构及保险费计算逻辑，设置等待期调整系数，等待期越长，保险公司面临的风险越小。
- 注：如果等待期介于上述相邻等待期之间，调整系数采用线性插值的方式计算。
- （五）给付比例调整系数
- 根据本保险的费率结构及保险费计算逻辑，设置给付比例调整系数，给付比例越高，保险公司面临的风险越大。
- 注：1.如果给付比例介于上述相邻给付比例之间，调整系数采用线性插值的方式计算。
- 2.对于一般医疗费用保险责任及重大疾病医疗费用保险责任，若被保险人以参加基本医疗保险身份投保，但未以参加基本医疗保险身份就诊并结算的，则给付比例为投保时约定给付比例的60%。例如：被保险人以参加基本医疗保险身份投保且投保时约定的给付比例为90%，出险后被保险人未以参加基本医疗保险身份就诊并结算，则实际给付比例为90%×60%=54%。
- 3.对于恶性肿瘤特定药品费用保险责任，若被保险人以参加基本医疗保险身份投保，但出险后所购药品在基本医疗保险药品目录内且未经过基本医疗保险报销的，则给付比例为投保时约定给付比例的60%。例如：被保险人以参加基本医疗保险身份投保且投保时约定的给付比例为90%，出险后所购药品在基本医疗保险药品目录内且未经过基本医疗保险报销，则实际给付比例为90%×60%=54%。

- 三、费率调整系数
- 费率调整系数为以下各调整系数的乘积。
- （一）投保人数规模调整系数
- 投保人数规模的大小对业务整体赔付的稳定性以及业务维护费用的分摊均产生影响。相比较而言，投保人数规模越大，其业务赔付数据越稳定，业务维护人均成本越低；反之亦然。
- （二）地区状况调整系数
- 对地区的经济发展水平、医疗卫生条件、环境质量等进行综合分析。相比较而言，经济越发达、医疗卫生条件越好、环境质量越高的地区，发生就医的概率会相对越低。
- （三）被保险人生活方式调整系数
- 被保险人的生活方式与其身体健康状况有较大关联关系，一般情况下，科学的生活方式会带来较低的疾病发生率。
- （四）被保险人健康状况调整系数
- 被保险人的健康状况与其就医的概率密切相关，一般情况下，身体健康的被保险人在保险期间内发生就医的概率较低。
- （五）渠道类型调整系数
- 根据不同业务渠道的费用差异确定合理的调整系数。
- （六）预期/经验赔付率调整系数

- 四、保险费计算公式
- 每人一般医疗费用和重大疾病医疗费用保险费=一般医疗费用和重大疾病医疗费用年基准保险费×产品参数调整系数×费率调整系数
- 每人恶性肿瘤特定药品费用保险费=恶性肿瘤特定药品费用年基准保险费×产品参数调整系数×费率调整系数
- 每人质子重离子医疗费用保险费=质子重离子医疗费用年基准保险费×产品参数调整系数×费率调整系数
- 每人重大疾病住院津贴保险费=重大疾病住院津贴年基准保险费×（每日重大疾病住院津贴金额÷100）×产品参数调整系数×费率调整系数
- 每人重大疾病异地转诊交通费用保险费=重大疾病异地转诊交通费用年基准保险费×产品参数调整系数×费率调整系数
- 每人保险费=每人一般医疗费用和重大疾病医疗费用保险费+每人恶性肿瘤特定药品费用保险费+每人质子重离子医疗费用保险费+每人重大疾病住院津贴保险费+每人重大疾病异地转诊交通费用保险费
- 总保险费=∑每人保险费
