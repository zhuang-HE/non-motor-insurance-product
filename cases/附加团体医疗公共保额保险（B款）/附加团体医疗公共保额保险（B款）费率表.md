# 附加团体医疗公共保额保险（B款）费率表

太平科技保险股份有限公司
附加团体医疗公共保额保险（B款）费率表

- 一、基准费率（年费率）
- 1.5%

- 二、费率调整系数
- 费率调整系数为以下各调整系数之积。
- （一）公共保额个人赔偿限额调整系数
- 本附加险的保险费计算基础是公共保额个人赔偿限额，该赔偿限额越高，保险费充足度越高。
- （二）公共保额保险金额调整系数
- 根据本附加险的费率结构及保险费计算逻辑，设置公共保额保险金额调整系数，该保险金额越高，保险公司面临的风险越大。
- （三）免赔额调整系数
- 根据本附加险的费率结构及保险费计算逻辑，设置免赔额调整系数，免赔额越高，保险公司面临的风险越低。
- （四）给付比例调整系数
- 根据本附加险的费率结构及保险费计算逻辑，设置给付比例调整系数，给付比例越高，保险公司面临的风险越大。
- 注：如果给付比例介于上述相邻给付比例之间，调整系数采用线性插值的方式计算。
- （五）被保险人拥有费用补偿型医疗保险情况调整系数
- 本附加险为费用补偿型医疗保险，在被保险人拥有其他费用补偿型医疗保险时，本附加险项下的风险相对会较低。
- （六）投保人数规模调整系数
- 投保人数规模的大小对业务整体赔付的稳定性以及业务维护费用的分摊均产生影响。相比较而言，投保人数规模越大，其业务赔付数据越稳定，业务维护人均成本越低；反之亦然。
- （七）地区状况调整系数
- 对地区的基础设施水平、医疗卫生条件、环境质量等进行综合分析。相比较而言，基础设施越完善、医疗卫生条件越好、环境质量越高的地区，发生就医的概率会相对越低。
- （八）被保险人生活方式调整系数
- 被保险人的生活方式与其身体健康状况有较大关联关系，一般情况下，科学的生活方式会带来较低的就医概率。
- （九）被保险人健康状况调整系数
- 被保险人的健康状况与其就医的概率密切相关，一般情况下，被保险人身体越健康，其在保险期间内发生就医的概率越低。
- （十）渠道类型调整系数
- 根据不同业务渠道的费用差异确定合理的调整系数。
- （十一）预期/经验赔付率调整系数

- 三、保险费计算公式
- 每人保险费=基准费率×公共保额个人赔偿限额×费率调整系数
- 总保险费=∑每人保险费

- 四、短期费率表（按年费率的百分比计算）
注：1.保险期间满一个月不满两个月的，按两个月计算，以此类推。
2.保险期间不满一个月的，按日计算短期费率，每日短期费率为年费率的1%，最高不超过20%。
