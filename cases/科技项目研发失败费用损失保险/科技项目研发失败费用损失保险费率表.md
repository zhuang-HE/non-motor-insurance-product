# 科技项目研发失败费用损失保险费率表

太平科技保险股份有限公司
科技项目研发失败费用损失保险费率表

- 一、基准费率
- 13%

- 二、费率调整系数
- 费率调整系数为以下各系数的乘积。
- （一）保险金额调整系数
- 保险费的计算基础为保险金额，保险金额越高，保险费充足度越高。
（二）保险期间调整系数
保险期间越长，保险人面临的风险暴露期间就越长，承担的风险也就越大。
（三）扩展灾因调整系数
根据实际业务承保过程中扩展灾因的情况按照下表确定合理的调整系数。
注：扩展灾因，指除条款第五条列明的第（一）至（四）项造成科技项目研发失败的原因外，保险人根据条款第五条第（五）项的约定额外承保了其他造成科技项目研发失败的原因。
（四）科技项目研发人员背景状况调整系数
通常情况下，研发人员的学历越高，相关经验越丰富，科技项目研发成功的概率也会越高。
注：根据科技项目研发人员的平均背景状况确定相应系数。
（五）被保险人项目管理状况调整系数
通常情况下，被保险人的项目管理机制越完善，以往实施的项目管理案例越丰富，科技项目研发成功的概率也会越高。
（六）免赔调整系数
- （七）业务渠道类型调整系数
- 根据不同业务渠道的费用差异确定合理的调整系数。

- 三、保险费计算公式
- 保险费=基准费率×保险金额×费率调整系数
