# 保险条款用语标准化矩阵

> 规范各险种条款撰写中的标准表述，确保条款语言的统一性和准确性

## 一、条款章节标准用语

| 章节名称 | 标准表述规范 |
|---------|------------|
| 总则 | 第一条（订立目的与依据）、第二条（定义与解释） |
| 保险标的 | 第三条（保险标的范围） |
| 保险责任 | 第四条（保险责任）、第五条（赔偿金额计算） |
| 责任免除 | 第六条（责任免除）、第七条（特殊免责情形） |
| 保险金额与免赔额 | 第八条（保险金额确定）、第九条（免赔额设定） |
| 保险期间 | 第十条（保险期间）、第十一条（续保条款） |
| 投保人/被保险人义务 | 第十二条（如实告知）、第十三条（安全维护）、第十四条（出险通知） |
| 赔偿处理 | 第十五条（索赔材料）、第十六条（核赔时限）、第十七条（支付时限） |
| 代位追偿 | 第十八条（代位追偿权） |
| 争议处理 | 第十九条（仲裁条款）、第二十条（诉讼管辖） |
| 释义 | 第二十一条（术语定义） |

## 二、保险责任表述标准

### 2.1 责任类型表述

| 责任类型 | 标准表述 |
|---------|---------|
| 财产损失 | "对因下列原因造成保险标的的损失，负责赔偿" |
| 第三者责任 | "对被保险人在保险期间内造成第三者的人身伤亡或财产损失，负责赔偿" |
| 产品责任 | "对因产品存在缺陷造成使用者或第三者的人身伤害或财产损失，负责赔偿" |
| 雇主责任 | "对被保险人雇用的员工在受雇期间因工作遭受意外事故，负责赔偿" |
| 公众责任 | "对被保险人在经营场所内因意外事故造成第三者人身伤亡或财产损失，负责赔偿" |

### 2.2 赔偿限额表述

| 情形 | 标准表述 |
|------|---------|
| 累计赔偿限额 | "在保险期间内，保险人对所有赔偿请求的累计赔偿金额不得超过本保险合同载明的累计赔偿限额" |
| 每次事故限额 | "对每次事故的赔偿金额不得超过本保险合同载明的每次事故赔偿限额" |
| 每人赔偿限额 | "对每个受害人的赔偿金额不得超过本保险合同载明的每人赔偿限额" |
| 财产损失限额 | "对每次事故中第三者财产损失的赔偿金额不得超过本保险合同载明的每次事故财产损失赔偿限额" |

## 三、责任免除表述标准

### 3.1 通用免责条款

| 免责类型 | 标准表述 |
|---------|---------|
| 战争暴动 | "战争、敌对行动、军事行为、武装冲突、罢工、暴乱、恐怖活动" |
| 核风险 | "核辐射、核爆炸、核污染及其他放射性污染" |
| 不可抗力 | "自然灾害：地震、火山爆发、海啸等不可抗力" |
| 故意行为 | "被保险人的故意行为" |
| 违法犯罪 | "被保险人从事违法犯罪活动" |
| 精神损失 | "精神损害赔偿" |
| 间接损失 | "间接损失、罚款、罚金及惩罚性赔偿" |

### 3.2 财产险免责

| 免责类型 | 标准表述 |
|---------|---------|
| 自然损耗 | "保险标的的自然损耗、自然磨损、自然蒸发" |
| 内在缺陷 | "保险标的的内在缺陷、霉变、氧化" |
| 保管不当 | "被保险人未按照国家有关消防、安全使用规范管理保险标的" |
| 贬值 | "保险标的的市场价值贬落" |

### 3.3 责任险免责

| 免责类型 | 标准表述 |
|---------|---------|
| 工伤保险 | "被保险人的雇员因其工伤、职业病、被依法认定工伤、被评残，依据《工伤保险条例》或其他法律规定应由工伤保险基金支付的费用" |
| 已有疾病 | "被保险人雇员因疾病、分娩、流产、人工终止妊娠及因此产生的相关费用" |
| 行政处罚 | "罚款、罚金及惩罚性赔偿" |
| 合同责任 | "因合同的约定而承担的合同责任，除非该责任依法应由被保险人承担" |

## 四、赔偿处理表述标准

### 4.1 索赔材料

| 材料类型 | 标准表述 |
|---------|---------|
| 基础材料 | "保险单正本、索赔申请、损失清单、费用发票" |
| 事故证明 | "事故证明材料：包括但不限于警方报告、消防证明、气象证明等" |
| 身份证明 | "被保险人及受益人的身份证明" |
| 关系证明 | "与损失相关的权属证明、合同协议等" |
| 医疗材料 | "医院诊断证明、病历资料、医疗费用清单、伤残鉴定报告" |
| 死亡材料 | "死亡证明、户口注销证明、受益人身份证明" |

### 4.2 核赔与支付

| 环节 | 标准时限 | 标准表述 |
|------|---------|---------|
| 立案受理 | 3个工作日 | "保险人收到索赔申请后，应当在3个工作日内完成立案受理" |
| 资料补正 | 一次性通知 | "保险人应当在收到完整索赔材料后一次性通知被保险人需要补充的材料" |
| 核赔确认 | 30日 | "保险人收到完整索赔材料后，应当在30日内完成核定" |
| 支付赔款 | 10日 | "保险人应当在与被保险人达成赔偿协议后10日内支付赔款" |
| 拒赔通知 | 30日 | "保险人应当自做出核定之日起3日内向被保险人发出拒赔通知书并说明理由" |

## 五、费率与保费表述标准

| 项目 | 标准表述 |
|------|---------|
| 费率表 | "详见本保险合同附带的费率表" |
| 保费计算 | "保费 = 基准保费 × 费率调整系数" |
| 费率因子 | "费率调整系数由以下因子综合确定：风险等级、行业类别、投保规模、历史赔付记录等" |
| 首期保费 | "保险合同自投保人按照约定支付保费之日起生效" |
| 续期保费 | "投保人应当按照约定及时支付续期保费，逾期未支付的，保险人可给予30日宽限期" |

## 六、特殊产品条款用语

### 6.1 保证保险用语

| 用语类型 | 标准表述 |
|---------|---------|
| 保证责任 | "保险人按照本保险合同约定，在被保证人未按照基础合同约定履行义务且超过约定期限时，承担赔偿责任" |
| 索赔条件 | "权利人向保险人提出索赔时，应当提供基础合同、违约证明、损失证明等材料" |
| 代位追偿 | "保险人赔付后，有权代位行使被保险人对被保证人的追偿权" |

### 6.2 工程保险用语

| 用语类型 | 标准表述 |
|---------|---------|
|、建设期 | "保险期间自保险工程在工地动工或材料设备运抵工地之日起开始" |
| 试车期 | "在试车、考核和保证期内发生的保险责任范围内的事故，保险人负责赔偿" |
| 交叉责任 | "本保险条款不控制、不减少、也不影响被保险人相互之间已有的或可能产生的任何交叉责任" |

### 6.3 责任保险特殊用语

| 用语类型 | 标准表述 |
|---------|---------|
| 第三者 | "本保险合同所称第三者是指被保险人及其雇员以外的人" |
| 意外事故 | "指不可预料的、突然发生的、非本意的、非疾病的外来事故" |
| 既往症 | "指在保险合同生效前被保险人已患有的疾病或已有症状" |
| 等待期 | "自保险合同生效之日起的一定期限内（除另有约定外为30日），被保险人因疾病支出的医疗费用，保险人不负责赔偿" |

## 七、条款修订标识规范

| 修订类型 | 标识方式 |
|---------|---------|
| 新增条款 | 以"【新增】"标识 |
| 删除条款 | 以"【删除】"标识，并在正式文本中移除 |
| 修订条款 | 以"【修订】"标识，并说明修订内容 |
| 解释条款 | 以"【释义】"标识 |

## 八、法律依据引用规范

| 法规类型 | 引用表述 |
|---------|---------|
| 法律 | "依据《中华人民共和国保险法》第XX条规定" |
| 行政法规 | "依据《XX条例》第XX条规定" |
| 司法解释 | "依据《最高人民法院关于XX的解释》第XX条规定" |
| 行业规范 | "依据中国银行保险监督管理委员会《XX办法》规定" |
