# 典型非车险产品案例库

## 一、企业财产保险案例

### 案例1：某制造企业财产综合险

**客户背景**：
- 企业类型：汽车零部件制造
- 注册资本：5000万元
- 员工人数：300人
- 年产值：2亿元

**保险方案**：
| 项目 | 内容 |
|------|------|
| 保险标的 | 厂房、机器设备、存货 |
| 保险金额 | 固定资产8000万+存货3000万 |
| 保险险种 | 财产综合险 |
| 保险责任 | 火灾、爆炸、自然灾害 |
| 免赔额 | 每次事故2万元或损失金额10%，以高者为准 |
| 保险费率 | 2.5‰ |
| 年保费 | 27,500元 |

**风险特点**：
- 车间有喷涂工艺，火灾风险较高
- 部分进口设备，重置价值高
- 存货周转快，价值波动大

**承保措施**：
- 要求安装自动喷淋系统
- 对喷涂车间单独约定免赔额
- 存货采用流动资产申报方式
- 安排溢额再保险

**理赔案例**：
2022年台风导致厂房受损，理赔金额180万元，赔付率654%

## 二、责任保险案例

### 案例3：某上市公司董责险(D&O)

**客户背景**：
- 公司类型：A股上市公司
- 行业：新能源
- 市值：500亿元
- 董事高管：15人

**保险方案**：
| 项目 | 内容 |
|------|------|
| 保险责任 | 董事高管责任 |
| 赔偿限额 | 每次事故及累计1亿元 |
| 保险期间 | 1年 |
| 追溯期 | 首次投保无追溯期，续保可扩展 |
| 免赔额 | 每次事故100万元 |
| 保险费率 | 0.8‰（按公司市值） |
| 年保费 | 400万元 |

**风险特点**：
- 监管环境趋严，诉讼风险上升
- 股东维权意识增强
- 并购活动频繁，整合风险

**理赔案例**：
2021年因信息披露问题被投资者集体诉讼，最终和解赔偿3500万元

## 三、工程保险案例

### 案例5：某地铁建设工程一切险

**项目背景**：
- 工程类型：地铁线路建设
- 线路长度：25公里
- 合同金额：120亿元
- 工期：5年

**保险方案**：
| 项目 | 内容 |
|------|------|
| 保险标的 | 工程主体、施工设备、临时设施 |
| 保险金额 | 120亿元 |
| 保险险种 | 建工一切险+第三者责任 |
| 第三者责任限额 | 每次事故5000万，累计2亿 |
| 免赔额 | 物质损失50万，第三者责任10万 |
| 保险费率 | 3‰ |
| 总保费 | 3600万元 |

**风险特点**：
- 地质条件复杂，盾构施工风险高
- 穿越市区，第三者责任风险大
- 工期长，不可控因素多

**承保措施**：
- 聘请专业风控工程师驻场
- 要求承包商提供履约保证
- 安排国际再保险支持
- 设置里程碑节点检查

## 五、产品责任险案例

### 案例8：某玩具出口商产品责任险

**客户背景**：
- 企业类型：玩具制造商
- 主要产品：儿童玩具
- 出口地区：美国、欧盟
- 年销售额：8000万美元

**保险方案**：
| 项目 | 内容 |
|------|------|
| 保险责任 | 产品缺陷导致的人身伤害和财产损失 |
| 赔偿限额 | 每次事故1000万美元，累计3000万 |

**风险特点**：
- 美加地区惩罚性赔偿风险高
- 儿童产品安全标准严格

**理赔案例**：

## 六、失败案例分析

### 案例9：某P2P平台信用保证险（失败案例）

**项目背景**：
- 平台类型：P2P网贷
- 承保时间：2015-2018年
- 保险责任：借款人违约保证

**失败原因**：
1. 风险识别不足：未充分评估平台风控能力
2. 道德风险：平台为追求规模放松审核
3. 累积风险：单平台承保金额过大
4. 监管变化：P2P行业整顿导致大规模违约

**损失情况**：
- 承保保费：5000万元
- 赔付金额：3.2亿元
- 赔付率：640%

**经验教训**：
- 信用保证险需严控底层资产质量
- 单平台风险暴露需设限
- 需建立动态风险监控机制
- 关注监管政策变化

## 七、创新产品案例

### 案例10：无人机第三者责任险

**产品背景**：
- 市场需求：无人机应用快速增长
- 监管要求：部分地区强制投保
- 传统产品：无专门险种

**产品设计**：
| 项目 | 内容 |
|------|------|
| 保险标的 | 无人机运营第三者责任 |
| 分类定价 | 按用途（消费级/商业级/工业级） |
| 按重定价 | 按无人机重量分级 |
| 赔偿限额 | 10万-1000万元多档可选 |
| 免赔额 | 1000-10000元 |
| 年保费 | 100-5000元 |

**创新点**：
- 首创无人机专门险种
- 引入飞手资质因子
- 与飞控系统数据对接
- 实现动态定价

## 八、案例使用说明

### 如何应用这些案例

1. **产品开发参考**
   - 了解同类产品的保障范围
   - 参考定价水平和费率结构
   - 借鉴风险管控措施

2. **客户方案设计**
   - 根据客户特点匹配案例
   - 参考保障方案设计
   - 预估保费水平

3. **风险评估学习**
   - 了解不同行业风险特点
   - 学习风险识别方法
   - 掌握承保管控要点

4. **理赔处理参考**
   - 了解典型赔案类型
   - 学习理赔处理经验
   - 总结风险防控教训
