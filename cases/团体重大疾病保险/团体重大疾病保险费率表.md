# 团体重大疾病保险费率表

太平科技保险股份有限公司
团体重大疾病保险费率表

- 一、基准保险费
单位：人民币元
- 注：基准保险费对应的重度疾病保险金额和轻度疾病保险金额均为1万元。

- 二、费率调整系数
- 费率调整系数为以下各系数的乘积。
- （一）等待期调整系数
- 等待期越长，保险人实际承担的风险越低。
- 注：等待期调整系数不适用于续保的情形。
- （二）投保人数规模调整系数
- 投保人数规模的大小对业务整体赔付的稳定性以及业务维护费用的分摊均产生影响。相比较而言，投保人数规模越大，其业务赔付数据越稳定，业务维护人均成本越低；反之亦然。
- （三）地区状况调整系数
- 对地区的经济发展水平、医疗卫生条件、环境质量等进行综合分析。相比较而言，经济越发达、医疗卫生条件越好、环境质量越高的地区，罹患重大疾病的概率相对会越低。
- （四）被保险人生活方式调整系数
- 被保险人的生活方式与其身体健康状况有较大关联关系，一般情况下，科学的生活方式会带来较低的疾病发生率。
- （五）渠道类型调整系数
- 根据不同业务渠道的费用差异确定合理的调整系数。
- （六）预期/经验赔付率调整系数

- 三、保险费计算公式
- 每人重度疾病保险费=重度疾病保险责任基准保险费÷10000×重度疾病保险金额×费率调整系数
- 每人轻度疾病保险费=轻度疾病保险责任基准保险费÷10000×轻度疾病保险金额×费率调整系数
- 每人保险费=每人重度疾病保险费+每人轻度疾病保险费
- 总保险费=∑每人保险费

- 四、短期费率表（按年保费的百分比计算）
注：1.保险期间满一个月不满两个月的，按两个月计算，以此类推。
2.保险期间不满一个月的，按日计算短期费率，每日短期费率为年费率的1%，最高不超过20%。
