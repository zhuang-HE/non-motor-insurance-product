太平科技保险股份有限公司
诉讼财产保全责任保险费率表

一、基准费率
0.3%

二、费率调整系数
费率调整系数为以下各调整系数的乘积。
（一）赔偿限额调整系数
（二）诉讼保全方式调整系数
（三）保全标的物调整系数
（四）案件清晰程度调整系数
（五）保全标的物权属情况调整系数
（六）业务渠道预期/经验赔付率调整系数

三、保费计算公式
保险费=赔偿限额×基准费率×费率调整系数


| 赔偿限额（万元）     | 调整系数     |
| ------------ | -------- |
| 500以下（含）     | 1.0-3.0  |
| 500-1000（含）  | 0.85-1.0 |
| 1000-3000（含） | 0.7-0.85 |
| 3000-5000（含） | 0.6-0.7  |
| 5000以上       | 0.5-0.6  |

| ------ | ------- |

| 查封、扣押  | 1.0-1.2 |
| 其它方式   | 1.2-1.5 |
