# 专利执行保险附加指定法律服务机构保险费率表

太平科技保险股份有限公司
专利执行保险附加指定法律服务机构保险费率表

本附加险条款为限制性条款，可按法律服务机构等级下调主险费率，最高下调幅度不超过50%。



| 法律服务机构等级 | 主险费率下调幅度 |
| --- | --- |
| 一级（专业服务能力强，以往相关案件胜诉概率超过70%） | 40%-50% |
| 二级（专业服务能力中等，以往相关案件胜诉概率介于50%至70%之间） | 20%-40% |
| 三级（专业服务能力较弱，以往相关案件胜诉概率低于50%） | 0-20% |
