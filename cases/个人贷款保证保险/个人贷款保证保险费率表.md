# 个人贷款保证保险费率表

太平科技保险股份有限公司
个人贷款保证保险费率表

- 一、基准费率
- 7%

- 二、费率调整系数
- 费率调整系数为以下各系数的乘积。
- （一）贷款期限调整系数
- （二）抵/质押、担保比例调整系数
- （三）免赔率调整系数
- （四）金融机构历史坏帐率调整系数
- 应综合考虑合作金融机构信用环境状况，根据不同合作金融机构的历史坏帐率情况选用不同的系数。
- 注：金融机构历史坏账率=近三年平均贷款坏账率×0.4+最近一年贷款坏账率×0.6
- （五）还款能力调整系数
- 若借款人采用分期还款的方式，可使用此系数。
- （六）借款人信用状况调整系数
- 注：借款人信用状况主要参考中国人民银行的个人征信报告，以借款人的信用记录、贷款逾期、信用卡逾期、缴费信息等内容综合考评。
- （七）渠道系数
- 根据渠道费用差异选用合理系数。
- （八）预期/经验赔付情况调整系数

- 三、保险费计算公式
- 保险费=保险金额×基准费率×费率调整系数

- 四、退保系数
- 未满期保险费＝保险费×退保系数
- 其中退保系数按照保险合同已生效月份数占保险期间月份数的比例确定，确定方式见下表，保险合同已生效期间不足一个月的部分按一个月计算。
