# 航班延误保险费率表

太平科技保险股份有限公司
航班延误保险费率表

一、基准费率：
10.0%

二、费率调整系数
费率调整系数为以下各调整系数之积。
（一）航班延误起赔时间调整系数
（二）延误时间计算类型调整系数
（三）承运人类型调整系数
注：主要航空公司指根据中国民用航空局发布的《2018年民航行业发展统计公报》中定义的主要航空公司，包括南航、国航、东航、海南、深圳、四川、厦门、山东、上海、天津等十家航空公司。
（四）航班经停调整系数
（五）销售渠道调整系数
（六）渠道规模调整系数
（七）出行月份调整系数
（八）航线类别调整系数
（九）预期/经验赔付率调整系数

三、保险费计算
保险费=保险金额×基准费率×费率调整系数
