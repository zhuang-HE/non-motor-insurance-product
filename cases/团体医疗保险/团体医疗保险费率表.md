# 团体医疗保险费率表

太平科技保险股份有限公司
团体医疗保险费率表

- 一、基准费率
- （一）住院医疗保险责任：5.6%
- （二）门（急）诊医疗保险责任：12.6%

- 二、费率调整系数
- 费率调整系数为以下各系数的乘积。
- （一）保险金额调整系数
- 本保险为费用补偿型医疗保险，保险费计算基础为保险金额，保险金额越高，保险费充足度越高。
- 1.住院医疗保险金额调整系数（住院医疗保险责任适用）
- 2.门（急）诊医疗保险金额调整系数（门（急）诊医疗保险责任适用）
- （二）免赔额调整系数
- 1.住院医疗免赔额调整系数（住院医疗保险责任适用）
- 2.门（急）诊医疗免赔额调整系数（门（急）诊医疗保险责任适用）
- （三）给付比例调整系数
- 1.住院医疗保险金给付比例调整系数（住院医疗保险责任适用）
- 2.门（急）诊医疗保险金给付比例调整系数（门（急）诊医疗保险责任适用）
- （四）等待期调整系数
- 等待期越长，保险人实际承担的风险越低。
- （五）被保险人年龄调整系数
- 被保险人就医的概率与其年龄存在较大的关联关系，一般情况下，就医的概率随着年龄的增长具有先降后升的特征。
- （六）投保人数规模调整系数
- 投保人数规模的大小对业务整体赔付的稳定性以及业务维护费用的分摊均产生影响。相比较而言，投保人数规模越大，其业务赔付数据越稳定，业务维护人均成本越低；反之亦然。
- （七）地区状况调整系数
- 对地区的经济发展水平、医疗卫生条件、环境质量等进行综合分析。相比较而言，经济越发达、医疗卫生条件越好、环境质量越高的地区，发生就医的概率相对会越低。
- （八）被保险人生活方式调整系数
- 被保险人的生活方式与其身体健康状况有较大关联关系，一般情况下，科学的生活方式会带来较低的疾病发生率。
- （九）被保险人拥有费用补偿型医疗保险情况调整系数
- 本保险为费用补偿型医疗保险，在被保险人拥有其他费用补偿型医疗保险时，本保险项下的风险相对会较低。
- （十）渠道类型调整系数
- 根据不同业务渠道的费用差异确定合理的调整系数。
- （十一）预期/经验赔付率调整系数

- 三、保险费计算公式
- 每人住院医疗保险费=住院医疗保险责任基准费率×住院医疗保险金额×费率调整系数
- 每人门（急）诊医疗保险费=门（急）诊医疗保险责任基准费率×门（急）诊医疗保险金额×费率调整系数
- 每人保险费=每人住院医疗保险费+每人门（急）诊医疗保险费
- 总保险费=∑每人保险费

- 四、短期费率表（按年保费的百分比计算）
注：1.保险期间满一个月不满两个月的，按两个月计算，以此类推。
2.保险期间不满一个月的，按日计算短期费率，每日短期费率为年费率的1%，最高不超过20%。
