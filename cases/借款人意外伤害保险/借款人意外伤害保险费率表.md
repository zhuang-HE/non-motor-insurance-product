# 借款人意外伤害保险费率表

太平科技保险股份有限公司
借款人意外伤害保险费率表

- 基准费率：
0.5%

二、费率调整系数
费率调整系数为以下各调整系数的乘积。
1. 规模调整系数

2. 职业类别调整系数
注：职业类别参考《太平科技保险意外险职业分类表》。

3. 被保险人年龄调整系数

4. 预期/经验赔付率调整系数

三、保险费计算
保险费=保险金额×基准费率×费率调整系数

四、短期费率表
注：1.保险期间满一个月不满两个月的，按两个月计算，以此类推。
2.保险期间不满一个月的，按日计算短期费率，每日短期费率为年费率的1%，最高不超过20%。
