# 政府采购合同售后履约（保证金）保证保险费率表

太平科技保险股份有限公司
政府采购合同售后履约（保证金）保证保险费率表

- 一、基准费率
- 2.0%

- 二、费率调整系数
- 费率调整系数为以下各系数的乘积。
- （一）保险期间调整系数
- （二）保险金额调整系数
（三）被保险人类型调整系数
- （四）抵/质押、担保比例调整系数
- （五）采购标的类别调整系数
- （六）免赔率调整系数
（七）投保人年度营业收入调整系数
（八）投保人资产负债率调整系数
- （九）渠道类型调整系数
- 根据渠道费用差异选用合理系数。
- （十）预期/经验赔付情况调整系数

- 三、保险费计算公式
- 保险费=保险金额×基准费率×费率调整系数
