# 无人机损失保险费率表

太平科技保险股份有限公司
无人机损失保险费率表

- 一、基准费率
- 16.0%

- 二、费率调整系数
- 费率调整系数为以下各系数的乘积。
- （一）无人机类型调整系数
- （二）保险金额调整系数
（三）无人机用途调整系数
（四）无人机机龄调整系数
（五）无人机使用频率调整系数
（六）无人机驾驶员驾龄调整系数
- （七）免赔率调整系数
- （八）渠道类型调整系数
- 根据渠道费用差异选用合理系数。
- （九）预期/经验赔付情况调整系数

- 三、保险费计算公式
- 保险费=保险金额×基准费率×费率调整系数

- 四、短期费率表
注：保险期间不足1 个月的，按1 个月计算；保险期间在1 个月以上，不足2 个月的，按2个月计算；保险期间在2 个月以上，不足3 个月的，按3 个月计算，依此类推。
