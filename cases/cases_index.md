## 案例一：科技成果转化费用损失保险（2025版）

**目录位置**: `cases/太平科技保险股份有限公司科技成果转化费用损失保险（2025版）/`

### 产品概述
- **险种定位**: 专门保障科技成果转化项目失败导致的费用损失
- **市场背景**: 2024年科技成果转化合同金额2269.1亿元，同比增长10%；66.1万项合同

### 保险标的（5类费用可选投保）
1. 直接消耗的材料、燃料和动力费用
2. 研发人员及外聘人员劳务费用
3. 模具、工艺装备开发费，设备调整费，样品检验费
4. 成果论证、评审、验收、评估等费用
5. 保险人认可的其他相关损失费用

### 投保人/被保险人
- 高校、科研机构、企业事业单位、社会组织
- 须承担科技成果转化项目实施、运营和管理职责
- 项目须符合国家战略方向或经政府部门认定

### 核心保险责任
> 科技成果转化项目经有关科技主管部门或保险人认可的机构**认定失败或未能达到预期成果而终止**，且被保险人承诺不继续开展或转让同一项目

### 主要责任免除
- 投保人/被保险人故意、重大过失或犯罪行为
- 战争、核污染、大气污染
- 地震、海啸
- 行政行为或司法行为
- 投保前已知的知识产权纠纷或方案缺陷
- 未经认证擅自终止项目
- **原材料价格上升或目标产品价格下降导致经济性不达预期**（市场风险免除）
- 政府财政资金已补偿的部分
- 间接损失

### 风险分析
| 风险类型 | 主要表现 |
|---------|---------|
| 技术不成熟风险 | 科技成果先进性、实用性不达标，转化失败 |
| 市场风险 | 决策者业务水平、配套技术和管理水平不足 |
| 操作风险 | 产品开发/备案/配置/推广/承保各环节人员失误 |

### 核保重点
- 科技成果转化项目技术成熟度审查
- 项目须符合国家战略方向
- 再保安排：单一风险保额较大时临分分出

### 费率结构
> 费率表为.doc格式，需LibreOffice转换后读取

## 案例二：科技企业专利执行保险（2025版）

**目录位置**: `cases/太平科技保险股份有限公司科技企业专利执行保险（2025版）/`

### 产品概述
- **险种定位**: 保障科技企业专利被侵权后的维权费用
- **市场背景**: 2024年国内有效发明专利475.63万件；全国法院新收专利案件44,392件

### 保险标的
- 保险单列明的专利权

### 投保人/被保险人
- 符合《专利法》规定并获专利授权的科技企业、研究机构、高校、社会组织

### 核心保险责任（期内索赔制）
**第一责任（调查费用）**: 第三方首次侵权，被保险人为取证进行调查，且于保险期间内立案受理，赔偿立案前的：
- 调查费、公证费、交通费、住宿费（300元/人·天）、伙食补助（50元/人·天）

**第二责任（法律费用）**: 就上述事故在承保区域内首次提起诉讼/仲裁/行政处理，赔偿：
- 律师费、仲裁/诉讼费/行政处理费及保险人书面同意的其他费用

### 主要责任免除
- 故意行为、犯罪行为、重大过失
- 经被保险人许可或授权实施的行为
- 战争
- 符合专利法规定的强制许可行为
- **非被保险人自主发明创造且独家实施的专利**（核心除外）
- 立案受理后发生的调查费用
- 购买涉嫌侵权产品费用
- 追溯期前已知或已提起诉讼的纠纷

### 赔偿结构（分项限额）
- 调查费用每件专利赔偿限额
- 法律费用每件专利赔偿限额
- 调查费用累计赔偿限额
- 法律费用累计赔偿限额

### 费率结构
| 项目 | 基准费率 |
|------|---------|
| 调查费用 | **4.9%** |
| 法律费用 | **5.0%** |

**调整系数体系**:
- 累计赔偿限额调整系数（1倍→1.0，>100倍→0.3-0.4）
- 行业调整系数：电子信息(0.8-1.0) / 生物医药(1.2-1.4) / 航空航天(1.6-2.0) / 新材料(1.1-1.2) / 高技术服务(0.6-0.7) / 新能源(1.0-1.1)

## 案例三：科研仪器共享使用损坏补偿保险（2025版）

**目录位置**: `cases/太平科技保险股份有限公司科研仪器共享使用损坏补偿保险条款(2025版)/`

### 产品概述
- **险种定位**: 机器损失险改造版，专为科研仪器开放共享场景设计
- **市场背景**: 浙江省科技厅"共享保"项目，2026年底预计产生300-700万保费
- **改造逻辑**: 基于机器损失险，针对共享仪器使用场景优化

### 保险标的
- 载明于保险合同明细表中的，由被保险人**所有或合法占有并对外开放共享**的科研仪器

### 投保人/被保险人
- 科研仪器**所有者或管理者**（高校、研究院、科技机构）
- 须参与开放共享活动

### 核心保险责任（4类原因）
1. 设计、制造或安装错误、铸造和原材料缺陷
2. 离心力引起的断裂
3. 超负荷、超电压、碰线、电弧、漏电、短路、大气放电等电气原因
4. 除责任免除外的其他意外原因
> 含：防止/减少损失的必要合理费用

### 保险价值确定方式
- 重置价值（出险时）
- 实际价值（出险时）
- 其他（协商确定）

### 费率调整因子
- 保险金额
- 保险金额/承保时重置价值比例
- 保险标的平均年有效工作机时（小时）

### 风险控制亮点
- **保中核查**：被保险人须遵守安全法规，建立健全共享管理机制
- 保险人可对安全状况进行检查并提出书面建议

## 案例四：中试项目费用损失保险

**目录位置**: `cases/太平科技保险股份有限公司中试项目费用损失保险条款开发/`

### 产品概述
- **险种定位**: 保障中试项目失败导致的费用损失
- **市场背景**: 全国建设制造业中试平台超2400个；浙江20家工信部首批重点培育平台

### 中试项目定义
> 以实验室成果转化为主要内容，经政府相关部门**立项或备案**，或被保险人根据国家相关政策要求自主进行的**中间性试验项目**

### 保险标的（5类费用可选投保）
1. 直接消耗的材料、燃料和动力费用
2. 模具、工艺装备开发及制造费，装备安装调试费
3. 工作人员在中试项目运行期间的工资
4. 中试成果的论证、评审、验收、评估等费用
5. 保险人同意承保的其他费用

### 投保人/被保险人
- 高校、科研机构、企业事业单位、社会组织
- 须承担中试项目**实施、运营和管理职责**

### 核心保险责任
> 被保险中试项目经政府有关部门或保险人认可的机构、专家团队**认定未能达到预期成果而终止**

### 保险期间特殊设计
> 自保险责任起始日零时，至**中试项目实际终止日期**或**合同约定终止日**（两者以先发生为准）

### 风险分析
| 风险类型 | 具体表现 |
|---------|---------|
| 技术风险 | 技术可行性不确定、技术前景不明、效果不达预期 |
| 生产风险 | 设备与原材料适配性、生产安全风险 |
| 市场风险 | 市场接受度不确定、同类产品竞争 |
| 经营管理风险 | 资金链断裂、采购超支、人才流失、政策变动、法律纠纷 |

### 核保重点
- 中试项目**技术成熟度**审查
- 项目须符合国家战略方向或经政府部门认定

## 案例五：自动控制系统责任保险

**目录位置**: `cases/太平科技保险股份有限公司自动控制系统责任保险/`

### 产品概述
- **险种定位**: 责任险，保障自动控制系统故障致第三方人身/财产损失
- **市场背景**: 工业自动化/智能家居/智慧物流/低空经济/载人交通等场景

### 自动控制系统定义
> 通过智能电脑系统实现自动控制的操作系统（硬件+软件）
> - 硬件：感知/定位/计算/安全工具/智能外置硬件/数据传输/人机接口
> - 软件：环境感知/行为预测/规划控制/路径规划/高精地图/高精定位/操作系统/智能互联/人机交互/系统安全

### 投保人/被保险人
- 制造商、销售商、服务运营商
- 设备所有人、使用者
- 软件开发商、集成商
- 智能驾驶系统辅助平台提供商

### 核心保险责任（3类原因）
1. **系统自身故障**: 软件/硬件故障、软硬件不适配
2. **外部交互故障**: 定位系统被干扰、通信信号缺失、突发断电/断网
3. **其他系统故障或自动控制故障**
> 含：被提起诉讼/仲裁的法律费用

### 主要责任免除（重点关注）
- 故意/重大过失/犯罪行为
- **违反使用说明书或操作规程**
- 投保前已知缺陷
- **违法违规使用、超限使用**
- 战争、不可抗力
- **遭受网络攻击**（已明确责任免除）
- 自然灾害
- 违法生产/出售
- **知识产权侵犯**

### 责任限额结构（6项）
1. 累计赔偿限额
2. 每次事故赔偿限额
3. 每次事故财产损失赔偿限额
4. 每次事故人身伤亡赔偿限额
5. 每次事故每人人身伤亡赔偿限额
6. 每次事故每人医疗费用赔偿限额

### 风险控制重点
- 核查承保系统的检测检验结果（行业国标、行标比对）
- 确认承保系统为安全可控的产品
- 再保安排

## 案例六：无人机机身一切险及第三者责任险

**目录位置**: `cases/无人机机身一切险及第三者责任险/`

### 产品概述
- **险种定位**: 综合险（财产险+责任险），低空经济核心险种
- **市场背景**: 2024年中国民用无人机市场1108.49亿元，年均增长17.18%
- **目标**: 2026年保费50万元

### 保险标的（无人机范围）
- **机身险标的**: 无人机机身 + 出厂标准配置设备（不含后期加装非标配）
- **不含**: 地面控制设备/数据通信设备/第三方机载设备（拍摄/传感器等，需特别约定）
- 类型：固定翼/多旋翼/无人直升机

### 保险价值确认
| 机龄 | 保险价值 |
|------|---------|
| ≤1年（新机） | 重置价值（市场新机购置价） |
| >1年（旧机） | 实际价值（合同生效时/出险时市场价） |

### 核心保险责任
**机身险**: 飞行/滑行/地面，不论任何原因（责任免除除外），造成无人机及附件的**意外损失或损坏**（含施救费用）

**第三者责任险**: 无人机造成第三者**人身伤亡或财物损失**，依法应由被保险人承担的赔偿责任（含法律费用）

### 机身险特殊责任免除
- 自然磨损/渐进损坏/机械故障/内在缺陷（任何原因）
- **电路或电池损坏**导致的机身损失
- **进水/水浸/进油/进沙/异物**（鸟撞引起除外）
- **失踪/失窃/不可解释丢失**
- 搬运/运输/存储/维修/改装/拆机期间损失
- **桨叶/护桨等易损耗材单独损失**
- **反无人机装置干扰或控制**导致的损失
- 设计/制造/安装错误或材料缺陷引起的损失

### 第三者责任险特殊责任免除
- 被保险人雇员工作期间的人身/财产损失
- 任何参与操作人员的人身/财产损失
- 飞行信号干扰导致的非物理性损失
- 无人机对第三方机载设备造成的损失
- **喷洒/空投作业中喷洒物/空投物造成的损失**
- 侵犯个人隐私（操作失控/坠毁相撞引起的除外）

### 费率结构

**基准费率**:
| 险种 | 费率 |
|------|------|
| 机身险 | **16.0%** |
| 第三者责任险 | **1.0%** |

**调整系数（7项）**:
| 系数类型 | 范围 |
|---------|------|
| 无人机用途 | 警用0.6-0.9 / 农用1.2-1.4 / 其他0.9-1.5 |
| 机龄 | ≤1年:0.6-0.8 / 1-2年:0.8-0.9 / >5年:1.5-2.0 |
| 使用频率 | ≤5次/月:0.6-0.8 / >30次/月:1.1-1.3 |
| 操作员资质 | 有执照:0.9 / 无执照:1.0 |
| 免赔率 | ≤5%:1.1-1.2 / >30%:0.65-0.8 |
| 渠道类型 | 直销:0.8-1.0 / 其他:1.0-1.3 |
| 经验赔付率 | 根据实际赔付情况调整 |

### 核保重点（保前调查）
- 机器型号/规格/用途/购置时间/飞行时长/当前状况
- 历史维修状况/常规飞行路线
- 飞行操作人员资质情况
- 保险标的价值评估
- 持有人管理水平评估

## 通用结构规律（提炼自6个案例）

### 可行性研究报告固定结构（8章节）
1. **总体开发说明** - 监管依据（金发〔2025〕36号、银保监会令2021年第10号、保监发〔2016〕115号）
2. **开发背景/开发目的** - 市场数据+政策依据
3. **法律依据** - 现行法律框架
4. **主要特点** - 投被保人+标的+责任免除+保险金额/期间
5. **风险分析及风险控制** - 核保/再保/操作风险管控
6. **经营模式** - 销售/承保/理赔/业绩监控
7. （部分有）**保费测算/市场分析**
8. **总体结论**

### 条款固定结构（13章节）
1. 总则（合同组成+投被保人资格）
2. 保险标的（定义+范围）
3. 保险责任（主险责任）
4. 责任免除（原因免除+损失/费用免除）
5. 保险价值/保险金额/责任限额/免赔额（率）
6. 保险期间
7. 保险人义务
8. 投保人/被保险人义务（如实告知/通知义务/出险义务/单证提供）
9. 赔偿处理（代位求偿/重复保险/赔偿计算）
10. 争议处理和法律适用
11. 其他事项（解约/短期费率表）
12. 释义（附录）
13. 短期费率表（附录）

### 核保操作风险管控模板（6个方面）
每份报告均包含相同的操作风险管控措辞：
1. 产品开发阶段：流程管控+人员培训
2. 产品备案环节：标准化管理+复核机制
3. 产品披露环节：合规管控
4. 产品系统配置：流程管控+培训
5. 产品市场推广：销售人员培训+销售行为管理
6. 产品承保：规范核保流程+定期培训
