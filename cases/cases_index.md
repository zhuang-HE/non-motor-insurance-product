## 📋 文档层级说明

```
产品目录/
├── 条款文件       ← 保险产品主条款或附加条款
├── 费率表文件     ← 保费计算依据（主险/附加险均可能有）
├── 可研报告       ← 产品开发可行性研究报告（仅深度案例有）
└── 风险策略       ← 核保指引文件（仅部分产品有）
```

**判断主险 vs 附加险**：
- 附加险目录名/文件名通常含"附加"二字
- 主险可独立投保，附加险须与主险同时投保
