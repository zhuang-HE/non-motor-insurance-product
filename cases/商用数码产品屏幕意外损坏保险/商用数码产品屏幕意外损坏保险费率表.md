# 商用数码产品屏幕意外损坏保险费率表

太平科技保险股份有限公司
商用数码产品屏幕意外损坏保险费率表

- 一、基准费率
- 5%

- 二、费率调整系数
- 费率调整系数为以下各系数之乘积。
- （一）保险金额调整系数
- （二）保险金额占商用数码产品整机价格比例调整系数
- （三）单件商用数码产品保险期间内最高维修赔偿次数调整系数
- （四）免赔调整系数
- （五）销售渠道调整系数
- （六）渠道规模调整系数：
- （七）经验/预期赔付调整系数：

- 三、保费计算公式为：
- 保险费=基准费率×保险金额×费率调整系数

- 四、短期费率表
注：保险期间不足1 个月的，按1 个月计算；保险期间在1 个月以上，不足2 个月的，按2个月计算；保险期间在2 个月以上，不足3 个月的，按3 个月计算，依此类推。
