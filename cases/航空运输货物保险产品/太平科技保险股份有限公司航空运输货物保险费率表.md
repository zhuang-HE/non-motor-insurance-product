# 太平科技保险股份有限公司航空运输货物保险费率表

太平科技保险股份有限公司
航空运输货物保险费率表

- 一、基准费率
- （一）航空运输险：0.05%
- （二）航空运输一切险：0.07%

- 二、费率调整系数
- 费率调整系数为以下各调整系数之积。
- （一）保险标的调整系数
- 根据不同保险标的的受损难易程度等差异确定合理的调整系数。
- （二）运输航程调整系数
- （三）运输工具状况调整系数
- 根据运输工具状况确定合理的调整系数。一般而言，运输工具状况越差，发生保险事故的概率越高，反之亦然。
- （四）跨境情况调整系数
- 一般而言，相较于其他地区，启运地和目的地均为中国大陆更有利于保险人管控风险，反之亦然。
- （五）启运地、目的地调整系数
- 根据启运地、目的地的基础设施等状况确定合理的调整系数。一般而言，启运地、目的地的基础设施、运输管理状况越差，发生保险事故的概率越高，反之亦然。
- （六）货物包装状况调整系数
- 根据被保险货物的包装状况确定合理的调整系数。一般而言，货物包装越差，发生保险事故的概率越高，反之亦然。
- （七）被保险人类型调整系数
- 根据被保险人对于保险标的的责任等差异确定合理的调整系数。
- （八）每单保险金额调整系数
（九）业务渠道类型调整系数
根据不同业务渠道的费用差异确定合理的调整系数。
（十）预期/经验赔付情况调整系数

- 三、保费计算公式
- 航空运输险保险费=航空运输险保险金额×航空运输险基准费率×费率调整系数
- 航空运输一切险保险费=航空运输一切险保险金额×航空运输一切险基准费率×费率调整系数
