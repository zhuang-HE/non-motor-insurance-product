# 首版次软件质量安全责任保险费率表


太平科技保险股份有限公司
首版次软件质量安全责任保险费率表

一、基准费率
3%

二、费率调整系数
费率调整系数为以下各系数的乘积。
（一）累计责任限额调整系数
（二）每次事故第三者责任限额调整系数
（三）每次事故保险软件损失责任限额调整系数
（四）每次事故免赔额（率）调整系数
（五）追溯期调整系数
（六）软件技术风险调整系数
（七）软件开发者经验调整系数
（八）业务渠道类型调整系数
根据不同业务渠道的费用差异确定合理的调整系数。
（九）预期/经验赔付调整系数

三、保险费计算公式
保险费=累计责任限额×基准费率×费率调整系数
