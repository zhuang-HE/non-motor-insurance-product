# 产品关联索引 — 主险 · 附加险 · 费率表

> **用途**: 给定一个产品，可快速找到对应的条款文件、费率表文件，以及关联的附加险或主险
> **更新时间**: 2026-04-16

---

## 使用说明

本索引建立以下关联关系：

| 关联类型 | 说明 |
|---------|------|
| 主险 → 附加险 | 列出该主险下的所有附加险 |
| 附加险 → 主险 | 列出该附加险依附的主险 |
| 条款 → 费率表 | 条款文件与对应费率表文件的映射 |
| 条款 → 可研/风险策略 | 条款与配套文档的映射 |
