# 附加团体医疗公共保额保险条款

太平科技保险股份有限公司
附加团体医疗公共保额保险
条款

总则
在投保了太平科技保险股份有限公司各类团体健康险（以下简称“主险”）的基础上，投保人可以选择投保本附加险。本附加险合同未尽事项，以主险合同为准。主险合同与本附加险合同内容冲突之处，以本附加险合同为准。
本附加险合同由保险条款、投保单或者其他投保文件、保险单或者其他保险凭证、批单和其他有关约定书构成。主险合同中与本附加险条款相关的内容，亦为本附加险合同的组成部分。凡涉及本附加险合同的约定，均应采用书面形式。
本附加险合同的投保人、被保险人与主险合同一致。
除另有约定外，本附加险合同受益人为被保险人本人。
保险责任
在保险期间内，若按照主险合同计算的赔偿金额超出主险合同中被保险人的保险金额，对于超出部分，保险人按照本附加险合同的约定，扣除免赔额后，乘以给付比例，向被保险人给付保险金。
保险人在本附加险项下对同一被保险人一次或多次累计给付的公共保额保险金之和以本附加险合同约定的该被保险人的公共保额个人赔偿限额为限。当一次或多次累计给付的保险金之和达到该金额时，保险人对该被保险人的保险责任终止。
保险人在本附加险项下对团体全部被保险人一次或多次累计给付的公共保额保险金之和以本附加险合同约定的公共保额保险金额为限。当一次或累计给付的保险金之和达到该金额时，本附加险责任终止。
责任免除
除另有约定外，本附加险合同的责任免除与主险合同一致。
保险期间
除另有约定外，本附加险合同保险期间同主险合同一致。
保险金额、个人赔偿限额、免赔额、给付比例
本附加险合同的保险金额、个人赔偿限额、免赔额和给付比例由投保人和保险人双方约定，并在本附加险合同上载明。
保险金申请

申请主险合同保险金须提供的材料。
投保人的书面同意材料。
