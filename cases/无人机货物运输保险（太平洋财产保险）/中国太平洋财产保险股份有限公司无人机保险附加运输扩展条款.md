# 中国太平洋财产保险股份有限公司无人机保险附加运输扩展条款

中国太平洋财产保险股份有限公司
无人机保险附加运输扩展条款

依据本 保单 的条款、条件及除外责任，以及本附加险的下列条款、条件及除外责任，本 保单 扩展承保保单明细表中列明的无人机在保险期限及列明的地域范围内的运输过程中遭受的物质损失或损坏：
保险人仅在无其他有效保险的情况下承担赔偿责任。
除外责任
本保险保障不承保：
下列原因导致的损失或损坏：
被保险无人机在跑道被清除且适合起飞后在跑道上发生的损失。
飞行过程中被保险无人机的损失。
在被保险无人机着陆后到达跑到跑道出口之前。
修复磨损、渐进老化、固有缺陷、生锈或氧化、飞蛾或害虫破坏、变形或收缩等产生的费用。
下列情形产生的损失或损坏：
保养，维修，翻新，修复，修改或 任何类似的过程 ；
干燥，潮湿，暴晒或极端气温造成的损失，除非该损失因暴雨或火灾所致。
因无人看管的车辆造成的或其遭受的损失或损坏。
电气或机械故障。
折旧。
除了依据理 算确定 的、修复或重置被保险无人机的直接费用之外的任何其他损失。
处于租赁、有条件销售、收费或 或 其他产权关系中的被保险无人机遭受的损失。
遵照适飞指令 而增加的费用或成本。
附属物品和 / 或备件。
因渗漏、污染直接或间接导致的或可能导致的损失。
战争、侵略、外敌行动、敌对状态（无论宣战与否）、内战、叛乱、革命、起义、军事管制、军事政权、篡权或企图篡权引起的直接或间接的损失或损坏。
由任何政府或地方当局下令的没收、国有化、征用或破坏或损坏对财产造成损失或损坏。
i ） 财产或费用的损失、损坏或产生的任何间接损失。
ii) 任何性质的法律责任。
因下列原因直接或间接导致或促成：
由任何核燃料或从核燃料的燃烧产生的核废料所发出的放射性电离辐射或污染。
任何爆炸性核装置或和零部件的放射性、毒性、爆炸性或其它危害性。
核反应、核辐射或放射性污染。

前提条件
除非被保险人遵守了所有下列情形要求，否则保险人对本保单项下的索赔不承担赔偿责任：
勤勉尽职
被保险人应采取一切合理的注意和措施保护被保险无人机并使其处于良好状态。
运输
被保险人应确保被保险财产的装卸均由能够胜任的专业的装卸工 / 生产商指引。
安全及防护
无论何时被保险无人机处于无人看管状态，被保险人均应确保所有已告知保险人的火灾报警器和安全系统、锁具及其他物理保护措施可以被充分使用。
适用法律及司法管辖
本保单应受中华人民共和国的法律管辖，中华人民共和国法院对任何争议均拥有管辖权。
