# 创业项目失败费用损失保险费率表

太平科技保险股份有限公司
创业项目失败费用损失保险费率表

- 一、基准费率
- 13%

- 二、费率调整系数
- 费率调整系数为以下各系数的乘积。
- （一）保险金额调整系数
（二）被保险人成立时长调整系数
（三）创业项目团队成员背景状况调整系数
注：根据团队成员的平均背景状况确定相应系数。
（四）创业项目融资情况调整系数
（五）免赔调整系数
- （六）业务渠道类型调整系数
- 根据渠道费用差异选用合理系数。
- （七）预期/经验赔付情况调整系数

- 三、保险费计算公式
- 保险费=基准费率×保险金额×费率调整系数

- 四、短期费率表
注：保险期间不足1 个月的，按1 个月计算；保险期间在1 个月以上，不足2 个月的，按2个月计算；保险期间在2 个月以上，不足3 个月的，按3 个月计算，依此类推。
