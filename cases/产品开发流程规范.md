## 二、阶段详解

### 阶段一：市场调研与可行性研究

**输出物**: `产品开发可行性研究报告.md`

| 步骤 | 任务 | 输出要求 | 模板位置 |
|------|------|---------|---------|
| 1.1 | 确定保险标的 | 明确标的合法性、可保性 | 条款板块矩阵 |
| 1.2 | 确定投保人/被保险人 | 资格审查、保险利益认定 | 总则章节 |
| 1.3 | 明确保险责任范围 | 主险/附加险责任设计 | 保险责任章节 |
| 1.4 | 撰写可研报告 | 开发背景、风险分析、风险管控 | 可研报告模板 |

**可研报告章节结构**:
```
# XX产品开发可行性研究报告

## 一、总体开发说明
- 监管依据
- 开发目的

## 二、开发背景
- 市场规模
- 政策支持
- 法律依据

## 三、产品主要特点
- 投保人/被保险人
- 保险责任
- 责任免除
- 保险金额/责任限额
- 保险期间

## 四、保险费率及保费测算
- 基准费率
- 费率调整系数
- 预期保费

## 五、风险分析及风险控制
- 逆选择风险
- 道德风险
- 操作风险
- 风控措施

## 六、经营模式
- 销售模式
- 承保流程
- 理赔流程
- 业绩监控

## 七、总体结论
```

### 阶段二：条款撰写

**输出物**: `XX保险条款.md`

| 步骤 | 任务 | 依据来源 | 注意事项 |
|------|------|---------|---------|
| 2.1 | 确定险种大类 | 产品编号.xlsx | 正确归类 |
| 2.2 | 套用条款板块 | 条款板块矩阵 | 通用+特有板块 |
| 2.3 | 撰写条款内容 | 参考已有条款 | 监管合规 |
| 2.4 | 内审校对 | 监管要求 | 避免歧义 |

**条款撰写检查清单**:
- [ ] 总则：合同组成、被保险人定义
- [ ] 保险标的：明确范围、除外标的
- [ ] 保险责任：主险责任、附加责任
- [ ] 责任免除：通用除外、专项除外
- [ ] 保险金额/赔偿限额：金额确定方式
- [ ] 免赔额（率）：免赔约定
- [ ] 保险期间：起止时间
- [ ] 义务条款：告知义务、安全义务
- [ ] 赔偿处理：赔偿方式、计算方法
- [ ] 代位追偿：权益转让
- [ ] 争议处理：法律适用、仲裁/诉讼

## 四、质量控制

### 4.1 自检清单

**条款合规性检查**:
- [ ] 符合《财产保险公司保险条款和保险费率管理办法》
- [ ] 符合《财产保险公司保险产品开发指引》
- [ ] 条款无歧义、表述清晰
- [ ] 责任免除明确、无争议

**费率合规性检查**:
- [ ] 费率厘定合理、有依据
- [ ] 符合报行合一要求
- [ ] 费率表格式规范

### 4.2 审核流程

```
开发人撰写 → 部门内部审核 → 法务合规审核 → 精算审核 → 分管领导审批
```

## 五、自动化工具

### 5.1 产品创建脚本

```bash
python scripts/new_product.py <产品名称> <险种大类>
```

**功能**：
- 自动创建标准化的产品开发目录
- 生成条款、费率表、可研报告三个文档模板
- 自动填充头信息和基本框架

**示例**：
```bash
python scripts/new_product.py "无人机机身一切险" "特殊风险"
```

### 5.2 条款内容生成器

```bash
python scripts/clause_generator.py <产品名称> <险种大类>
```

**功能**：
- 根据险种大类自动生成完整的条款框架
- 包含12个险种大类的专业条款模板
- 自动生成保险标的、责任、免责等核心内容

**支持的险种大类**：
- 企财险、货运险、责任险、保证险
- 意外险、健康险、家财险、工程险
- 船舶险、信用险、特殊风险、其他

### 5.3 产品索引构建

```bash
python scripts/build_product_index.py
```

**功能**：
- 自动扫描cases目录
- 按险种大类分类所有产品
- 生成product_index.md索引文件
