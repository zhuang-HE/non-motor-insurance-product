# 建设工程施工合同履约保证保险费率表

太平科技保险股份有限公司
建设工程施工合同履约保证保险费率表

- 一、基准费率
- 1.1%

- 二、费率调整系数
- 费率调整系数为以下各系数的乘积。
- （一）投保人资质等级调整系数
- 注：投保人资质等级根据住房和城乡建设部印发的《建筑业企业资质标准》、《施工总承包企业特级资质标准》等文件确定。
- （二）建设工程项目性质调整系数
- （三）建设工程项目类型调整系数
- （四）保险金额调整系数
- （五）抵/质押、担保比例调整系数
- （六）免赔率调整系数
（七）投保人年度营业收入调整系数
（八）投保人资产负债率调整系数
- （九）渠道系数
根据渠道费用差异选用合理系数。
- （十）预期/经验赔付情况调整系数

- 三、保险费计算公式
- 保险费=保险金额×基准费率×费率调整系数

四、短期费率表
说明：保险期限不满一个月，按一个月计算；满一个月不满两个月，按两个月计算；以此类推。
