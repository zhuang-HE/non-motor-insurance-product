# 非车险产品开发技能

覆盖非车险产品从概念设计到条款输出的全流程开发工具。

## 产品开发流程（7步）

### 步骤1：确定保险标的概念
分析标的可保性（合法性、确定性、可估价性、可转移性、偶然性），确定标的分类和估价方法。

**科技保险/新兴领域专项**：开发首台套、知识产权、网络安全、绿色保险等产品时，需先掌握标的所属产业知识：
- `references/industry_knowledge.md` → 第一部分（科技保险核心产业知识）
- `references/industry_knowledge.md` → 第二部分（承保标的分类与研究方法）

**九大新兴产业与七大补充领域**：开发人形机器人、无人车、无人机、AI、生物医药等前沿领域保险时：
- `references/industry_knowledge.md` → 第四部分（新兴产业与未来产业知识体系）
  - 人形机器人、无人车、无人机、四足机器人、机械臂、工业机器人、AI、生物医药、医疗器械
  - 具身智能、集成电路、氢能、食品、保健食品、化妆品、原料药

**详细检查清单**: `references/development_checklist.md` → 步骤1

### 步骤2：明确投保人、被保险人身份
审查投保人/被保险人资格，认定保险利益类型（所有权/使用权/保管权/抵押权/合同权益/法定责任）。

**风险评估支撑**：需结合标的特性进行风险因子分析：
- `references/industry_knowledge.md` → 第二部分（风险因子分类体系）
- `references/industry_knowledge.md` → 第三部分（风险因子定性/计量方法）

**详细检查清单**: `references/development_checklist.md` → 步骤2

### 步骤3：初步明确保险责任范围
选择产品类型（基本险/综合险/一切险/特定风险险），设计主险和附加险责任，制定除外责任体系。

**标准体系参考**：条款设计须符合相关国家/行业/团体标准要求：
- `references/industry_knowledge.md` → 第一部分（GB/JR/T标准体系）
- `references/industry_knowledge.md` → 第二部分（风险环节识别方法）

**详细检查清单**: `references/development_checklist.md` → 步骤3

### 步骤4：撰写产品条款开发可行性研究报告
按8章节标准结构撰写，必须包含监管依据三段式引用和操作风险管控6个方面。

**风险定量分析**（可研报告第4-6章的精算支撑）需参考：
- `references/actuarial_methods.md` → 第一章（定价方法选择）+ 第五章（损失分布拟合）
- `references/actuarial_methods.md` → 第六章（数据质量控制与底稿管理）

**详细检查清单**: `references/development_checklist.md` → 步骤4

### 步骤5：撰写保险产品条款
按13章节+2附录标准结构撰写。**必须先查路由表定位最相似模板，再基于模板修改**。

#### 模板化撰写工作流

```
1. 路由定位 → 查上表找到最相似产品
2. 读取骨架 → 读取该产品的 条款.docx 作为13章骨架
3. 修改差异 → 仅修改以下5个差异化章节：
   - 第2章（保险标的）：定义新标的范围
   - 第3章（保险责任）：设计新责任条款
   - 第4章（责任免除）：查除外责任矩阵 + 产品特定除外
   - 第5章（金额/限额/免赔）：设计赔偿结构
   - 第12章（释义）：定义产品特有术语
4. 用语规范 → 按 development_checklist.md 5.4节标准用语对照表统一表述
5. 输出条款 → 完整13章节 + 短期费率表
```

**通用章节**（总则、义务、赔偿处理、争议处理、其他事项）可直接复用模板，仅修改被保险人类型和产品名称。

**详细检查清单**: `references/development_checklist.md` → 步骤5

**重点参考**：
- **5.4节（司法实践向用语规范）**：除外条款7项规范 + 责任条款5项规范 + 义务条款5项规范 + 禁用表述清单
- **5.6节（赔偿处理与防止不当得利）**：赔偿计算公式体系 + 14项完整性检查清单 + 司法败诉漏洞对照表

### 步骤6：产品定价和风险因子设定
- **定价方法**: 纯保费法、损失率法、Burning Cost法、类比法
- **风险因子体系**:
  - 财产险：建筑结构(25%)、消防设施(20%)、占用性质(25%)、建筑年代(15%)、安全管理(15%)
  - 责任险：行业类别(35%)、区域差异(20%)、历史记录(20%)、经营规模(15%)、限额免赔(10%)
  - **科技费用损失险**：技术成熟度(30%)、资金规模(20%)、认定级别(20%)、被保险人资质(15%)、保障比例(15%)
  - **专利执行险**：行业类别(25%)、限额倍数(25%)、专利类型(20%)、维持年限(15%)、企业规模(15%)
  - **科研仪器共享险**：仪器类型(25%)、年工作机时(25%)、保额/重置价值比(20%)、管理水平(15%)、安装环境(15%)
- **基准费率计算**: 费率 = 纯保费率 / (1 - 费用率 - 利润率)
- **调整因子应用**: 加权调整模型

## 使用方式

### 完整流程开发
按步骤1→2→3→4→5→6→7顺序执行：
1. 步骤1-7：参照 `references/development_checklist.md` 的检查清单，结合 `cases/` 案例库进行分析和撰写
2. 步骤7：使用 `python-docx` 直接生成 Word 文档，或调用 `scripts/step7_word_export.py` 的 `WordDocumentExporter` 类

### 单步执行
可单独执行任一步骤，如只需定价可执行步骤6，只需撰写条款可执行步骤5。

### 快速参考
直接查阅 references/ 和 cases/ 获取知识支持。

### 监管依据标准三段式（必须引用）
```
国家金融监督管理总局2025年9月30日印发的《关于加强非车险业务监管有关事项的通知》（金发〔2025〕36号）

原保监会印发的《财产保险公司保险产品开发指引》（保监发〔2016〕115号）
```

### 可行性研究报告固定结构（8章节）
1. 总体开发说明（监管依据）
2. 开发背景/开发目的（市场数据+政策依据）
3. 法律依据
4. 主要特点（投被保人+标的+责任免除+保险金额/期间）
5. 风险分析及风险控制（核保/再保/操作风险管控）
6. 经营模式（销售/承保/理赔/业绩监控）
7. （可选）保费测算/市场类似产品分析
8. 总体结论

### 条款固定结构（13章节）
总则 → 保险标的 → 保险责任 → 责任免除 → 保险价值/保险金额/免赔额 → 保险期间 → 保险人义务 → 投保人被保险人义务 → 赔偿处理 → 争议处理和法律适用 → 其他事项 → 释义 → 短期费率表

### 操作风险管控模板（6个方面，可行性报告必含）
1. 产品开发阶段：流程管控+人员培训
2. 产品备案环节：标准化管理+复核机制
3. 产品披露环节：合规管控
4. 产品系统配置：流程管控+培训
5. 产品市场推广：销售人员培训+销售行为管理
6. 产品承保：规范核保流程+定期培训
